# Navbar & Sidebar

The two layout atoms that compose the app shell: a horizontal `Navbar` for top-bar navigation and a vertical `Sidebar` for persistent side navigation, both following the Catalyst Headless UI pattern.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `src/components/playground/navbar` | `Navbar`, `NavbarSection`, `NavbarItem`, `NavbarSpacer`, `NavbarDivider`, `NavbarLabel` |
| `src/components/playground/sidebar` | `Sidebar`, `SidebarHeader`, `SidebarBody`, `SidebarFooter`, `SidebarSection`, `SidebarItem`, `SidebarLabel`, `SidebarHeading`, `SidebarDivider`, `SidebarSpacer` |
| `@heroicons/react` | Navigation icons |

## Notifies

None — navigation is handled via `href` props on items.

## Data Shapes

### Navbar Sub-components

```typescript
// NavbarItem accepts href for routing
interface NavbarItemProps {
  href: string;
  current?: boolean;
  'aria-label'?: string;
  children: React.ReactNode;
}
```

### Sidebar Sub-components

```typescript
// SidebarItem accepts href for routing
interface SidebarItemProps {
  href: string;
  current?: boolean;
  children: React.ReactNode;
}

// SidebarHeading renders a section label
interface SidebarHeadingProps {
  children: React.ReactNode;
}
```
