# VisibilityBadge

A colored icon-badge indicating the visibility tier of a metric in the runtime memory system; used in debug-mode overlays to show whether a metric is displayed, recorded, promoted, or private.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `MetricVisibility` | Union type driving the icon and color (`'display' \| 'result' \| 'promote' \| 'private'`) |
| Heroicons / Lucide icons | Eye, Check, Arrow, Lock icons per tier |

## Notifies

None — display-only component.

## Data Shapes

### Props

```typescript
interface VisibilityBadgeProps {
  visibility: MetricVisibility;
}

type MetricVisibility = 'display' | 'result' | 'promote' | 'private';
```

### Tier Legend

| Value | Color | Icon | Meaning |
|-------|-------|------|---------|
| `display` | green | Eye | Shown in main display |
| `result` | purple | Check | Recorded as a result |
| `promote` | blue | Arrow | Promoted to parent block |
| `private` | amber | Lock | Private to current block |
