# CollectionWorkoutsList

A filtered workout list for a collection category; wraps `QueriableListView` and `FuzzySearchQuery` so users can type in the search box to filter workouts by name.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `QueriableListView` | Filterable list rendering |
| `FuzzySearchQuery` | Fuzzy text matching |
| `nuqs` | URL search-param state for the query |
| `WorkoutItem` | Workout data shape |
| `playground/src/views/queriable-list/CollectionWorkoutsList` | Component implementation |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `onSelect` | Workout item clicked | `WorkoutItem` |

## Data Shapes

### Props

```typescript
interface CollectionWorkoutsListProps {
  items: WorkoutItem[];
  onSelect?: (item: WorkoutItem) => void;
}

interface WorkoutItem {
  id: string;
  name: string;
  category: string;
  content: string;
}
```
