# Review — Chromecast

The Chromecast-optimized layout for the post-workout review screen; rendered on the TV after a workout completes, driven by `WorkbenchDisplayState` JSON fixtures matching the exact wire format from `ChromecastProxyRuntime`.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `ReceiverReviewPanel` | Chromecast-optimized review panel |
| `WorkbenchDisplayState` | Wire-format data from `ChromecastProxyRuntime` |
| `@/panels/review-panel-chromecast` | Component implementation |

## Notifies

None — display-only panel for TV receiver.

## Data Shapes

### Props

```typescript
interface ReceiverReviewPanelProps {
  displayState: WorkbenchDisplayState;
}
```

States covered: SimpleRows, WithProjections, FranResults, AmrapResults.
