# CanvasPage

The unified layout shell used by every content route in the playground; supports a title-bar mode (sticky header + optional TOC sidebar) and a sections mode (StickyNavPanel + anchor-linked section blocks).

## Depends On

| System / Token | Usage |
|----------------|-------|
| `StickyNavPanel` | Section navigation in sections mode |
| `INavActivation` | Nav item type for section links |
| `src/panels/page-shells/CanvasPage` or equivalent | Component implementation |

## Notifies

None — layout shell; scroll behavior is managed internally.

## Data Shapes

### Props

```typescript
// Title-bar mode
interface CanvasPageTitleProps {
  title: string;
  subheader?: React.ReactNode;
  index?: React.ReactNode;     // TOC sidebar (visible at 3xl+)
  children: React.ReactNode;
}

// Sections mode
interface CanvasPageSectionsProps {
  sections: INavActivation[];
  children: React.ReactNode;
}
```
