# NoteEditor — Web

The desktop-viewport layout for the workout note editor; uses `StorybookWorkbench` at full screen width with the `EditorShellHeader` toolbar (Cast · Theme · Debug · Download · Reset).

## Depends On

| System / Token | Usage |
|----------------|-------|
| `StorybookWorkbench` | Editor shell with `EditorShellHeader` toolbar |
| `NoteEditor` | Monaco-based workout editor |
| `EditorShellHeader` | Cast · Theme · Debug · Download · Reset toolbar |

## Notifies

None at the template level — interactions are handled by the embedded editor.
