# HomeHero

The hero section for the playground home page; features a gradient wash, headline, subtitle, three feature cards, and CTA buttons that navigate using React Router.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `react-router-dom useNavigate` | CTA button navigation; provided by global `MemoryRouter` decorator |
| `CommitGraph` | Animated background contribution grid |
| `playground/src/components/HomeHero` | Component implementation |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| Route push | CTA button clicked | route path string |
