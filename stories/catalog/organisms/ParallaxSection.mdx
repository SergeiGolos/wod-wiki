# ParallaxSection

A scroll-driven section that uses `IntersectionObserver` to detect when each step enters the viewport and fires `onStepChange`; use it to build scroll-triggered narrative or parallax layouts.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `IntersectionObserver` | Step entry detection |
| `src/panels/page-shells/ParallaxSection` | Component implementation |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `onStepChange` | A step enters the viewport | `stepIndex: number` |

## Data Shapes

### Props

```typescript
interface ParallaxSectionProps {
  steps: React.ReactNode[];
  onStepChange?: (stepIndex: number) => void;
  threshold?: number;   // IntersectionObserver threshold (0–1)
  children?: React.ReactNode;
}
```

### Internal State

| State | Type | Initial | Purpose |
|-------|------|---------|---------|
| `activeStep` | `number` | `0` | Currently visible step index |
