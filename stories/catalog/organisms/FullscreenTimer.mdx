# FullscreenTimer

The core workout overlay rendered when a user starts a workout from the editor or tracker route; wraps `RuntimeTimerPanel` in a `FocusedDialog` portal and transitions to `ReviewGrid` when the workout completes.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `FocusedDialog` | Full-viewport portal overlay |
| `RuntimeTimerPanel` | Live timer and metric display |
| `ReviewGrid` | Post-workout results view |
| `WodBlock` | Workout block type providing the script |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `onClose` | Close (✕) button clicked | none |

## Data Shapes

### Props

```typescript
interface FullscreenTimerProps {
  block: WodBlock;
  autoStart?: boolean;
  onClose: () => void;
}
```

### Internal State

| State | Type | Initial | Purpose |
|-------|------|---------|---------|
| `phase` | `'timer' \| 'review'` | `'timer'` | Whether to show timer or post-workout review |
