# RuntimeTimerPanel

An in-overlay timer UI that creates its own `ScriptRuntime` via `RuntimeFactory` on mount; displays the active timer, metric overlays, and manages state transitions for a WOD block. Click **Start** to begin the runtime.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `RuntimeFactory` | Creates and wires a `ScriptRuntime` from a `WodBlock` |
| `ScriptRuntime` | Execution engine; created on mount, disposed on unmount |
| `TimerStackView` | Timer and exercise card display |
| `WodBlock` | Workout block providing the script |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `onComplete` | Workout finishes naturally | `ScriptRuntime` (for review) |
| `onClose` | Close button clicked | none |

## Data Shapes

### Props

```typescript
interface RuntimeTimerPanelProps {
  block: WodBlock;
  autoStart?: boolean;
  onComplete?: (runtime: ScriptRuntime) => void;
  onClose?: () => void;
}
```

### Internal State

| State | Type | Initial | Purpose |
|-------|------|---------|---------|
| `runtime` | `ScriptRuntime \| null` | `null` | Created by `RuntimeFactory` on mount |
