# Syntax

The WOD Wiki syntax reference page; uses `StorybookWorkbench` in readonly mode to display an interactive syntax guide with live parsing so users can see how workout scripts are parsed.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `StorybookWorkbench` | Full-screen editor shell |
| `NoteEditor` | Read-only Monaco editor with syntax highlighting |
| `JitCompiler` | Live parse results displayed alongside the script |

## Notifies

None — read-only reference page.
