# Planner

The full-page plan and editor view where users write and structure workouts using the WOD Wiki syntax with live preview and JIT compilation.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `PlanPanel` | Plan layout shell (`src/panels/plan-panel`) |
| `NoteEditor` | Monaco-based workout editor |
| `JitCompiler` | Live workout compilation for preview |

## Notifies

None at the page level — editor interactions are internal.
