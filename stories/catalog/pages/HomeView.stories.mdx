# HomeView

The application home page combining `HomeHero` (gradient banner + CTAs) with the home canvas page content and a workout list.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `HomeHero` | Hero banner template |
| `WorkoutItem` | Workout data shape for the list |
| `playground/src/pages/HomeView` | Page implementation |
| `react-router-dom useNavigate` | CTA navigation; provided by `MemoryRouter` |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| Route push | CTA or workout item clicked | route path |

## Data Shapes

### Props

```typescript
interface HomeViewProps {
  workoutItems?: WorkoutItem[];
}

interface WorkoutItem {
  id: string;
  name: string;
  category: string;
  content: string;
}
```
