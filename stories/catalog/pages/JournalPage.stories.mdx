# JournalPage

The daily journal entry page that loads workout content from IndexedDB by date-key; in stories the default template content is shown since IndexedDB is empty.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `IndexedDB` | Persists and loads journal entries by date |
| `NoteEditor` | Monaco-based workout editor |
| `playground/src/pages/JournalPage` | Page implementation |
| `react-router` | Route param: `/journal/:date` |

## Notifies

None — editor writes directly to IndexedDB on change.

## Data Shapes

### Props

```typescript
// Route-driven; no direct props
// Route: /journal/:date (YYYY-MM-DD)
```
