# JournalPageShell

The full-page note editor shell for writing and structuring workouts with live parsing; uses the same web editor shell as the Syntax reference stories with `EditorShellHeader` toolbar.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `StorybookWorkbench` | Full-screen editor shell with toolbar |
| `NoteEditor` | Monaco-based workout editor with live parsing |
| `EditorShellHeader` | Cast · Theme · Debug · Download · Reset toolbar |

## Notifies

None at the page level — the editor handles all interactions.

### Props

```typescript
// Driven by StorybookWorkbench; no external page props

  showToolbar?: boolean;
  readonly?: boolean;
  theme?: string;
}
```
