# WorkoutEditorPage

The main workout editor page that renders workout content from a collection in the note editor with collection-mode WOD commands (Run, Today, Plan).

## Depends On

| System / Token | Usage |
|----------------|-------|
| `NoteEditor` | Monaco-based workout editor |
| `WodCollectionItem` | Workout content passed via `mdContent` prop |
| `playground/src/pages/WorkoutEditorPage` | Page implementation |
| `FullscreenTimer` | Launched when Run is triggered |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| Run command | User clicks Run in the WOD block | launches `FullscreenTimer` overlay |
| Today command | User clicks Today | creates journal entry for today |
| Plan command | User clicks Plan | routes to planner |

## Data Shapes

### Props

```typescript
interface WorkoutEditorPageProps {
  mdContent?: string;       // static markdown content for the story
  collectionId?: string;
  itemId?: string;
}
```
