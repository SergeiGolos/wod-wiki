# TrackerPage

The workout performance tracking page that runs a workout from a pending runtime stored in the in-memory `pendingRuntimes` map; in stories it shows the error state since no real runtime is available.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `pendingRuntimes` | In-memory map of runtime id → `ScriptRuntime` |
| `TimerScreen` / `VisualStatePanel` | Tracker UI components |
| `playground/src/pages/TrackerPage` | Page implementation |
| `react-router` | Route param: `/tracker/:runtimeId` |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| Route push on complete | Workout finishes | `/review/:runtimeId` |

## Data Shapes

```typescript
// Route-driven; no direct props
// Route: /tracker/:runtimeId
```
