# StickyNavPanel

A sticky top-of-panel navigation bar where each item is an `INavActivation`; clicking dispatches the action through `executeNavAction`. Supports short and long item lists.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `INavActivation` / `NavActionDeps` | Action descriptor and dispatch dependencies |
| `executeNavAction` | Dispatches scroll or call actions |
| `src/panels/page-shells/StickyNavPanel` | Component implementation |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `executeNavAction(action, deps)` | Nav item clicked | `INavActivation` action |

## Data Shapes

### Props

```typescript
interface StickyNavPanelProps {
  items: INavActivation[];
  deps?: NavActionDeps;
  activeId?: string;
}
```

### Internal State

| State | Type | Initial | Purpose |
|-------|------|---------|---------|
| `activeId` | `string \| undefined` | `undefined` | Highlights the currently active section |
