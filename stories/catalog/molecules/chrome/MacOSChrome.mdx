# MacOSChrome

A macOS-style window chrome decoration for Storybook demos; wraps content in a window frame with traffic-light buttons, an optional title, a reset button, and custom header actions.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `playground/src/components/MacOSChrome` | Component implementation |
| Lucide `RefreshCw` icon | Optional reset button icon |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `onReset` | Reset button clicked | none |

## Data Shapes

### Props

```typescript
interface MacOSChromeProps {
  title?: string;
  onReset?: () => void;
  headerActions?: React.ReactNode;
  children: React.ReactNode;
}
```
