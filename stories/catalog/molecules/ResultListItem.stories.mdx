# ResultListItem

A workout result row with a timestamp column on the left, a status icon, and a title plus optional subtitle; used in journal day-scrolls and note-editor WOD-block result lists.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `src/components/results/ResultListItem` | Component implementation |
| Design tokens | Border, text, and icon colors |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `onClick` | User clicks the row | none |

### Props
