# SidebarAccordion

A collapsible section group used inside `NavSidebar` for L2 navigation items (e.g. Syntax sub-pages, collection groups); built on Headless UI Disclosure with animated chevron.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `@headlessui/react` Disclosure | Accessible open/close behavior |
| `SidebarItem` / `SidebarSection` | Children slot content |

## Notifies

None — open/close state is internal.

## Data Shapes

### Props

```typescript
interface SidebarAccordionProps {
  title: string;
  defaultOpen?: boolean;
  count?: number;        // optional badge showing item count
  children: React.ReactNode;  // typically SidebarItem list
}
```

### Internal State

| State | Type | Initial | Purpose |
|-------|------|---------|---------|
| `open` | `boolean` | `defaultOpen ?? false` | Disclosure open/close (managed by Headless UI) |
