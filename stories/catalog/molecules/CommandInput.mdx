# CommandInput

The search-input row used at the top of every command-palette variant: a leading search icon, a text input, and a trailing Escape badge.

> **Note:** This story uses a surrogate `CommandInputRow` component for isolated visual documentation. In production the input is `cmdk`'s `Command.Input`, which requires `Command.List` for accessible ARIA semantics.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `cmdk` | Production `Command.Input` (not used in story surrogate) |
| Lucide `Search` icon | Leading search icon |
| `ShortcutBadge` | Trailing `Esc` badge |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `onChange` | User types in the input | `string` — current value |

## Data Shapes

### Props

```typescript
interface CommandInputRowProps {
  value?: string;
  placeholder?: string;
  onChange?: (value: string) => void;
}
```

### Internal State

| State | Type | Initial | Purpose |
|-------|------|---------|---------|
| `value` | `string` | `''` | Controlled input value |
