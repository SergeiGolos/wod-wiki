# MetricTrackerCard

Floating metric bubbles that display session-total analytics; renders nothing when there is no active workout runtime in context.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `ScriptRuntimeProvider` / `ScriptRuntime` | Provides the active runtime context |
| `useOutputStatements()` | Reads the live output stream; session totals arrive as `'analytics'`-typed outputs emitted per segment by `AnalyticsEngine` |

## Notifies

None — display-only component.

## Data Shapes

### Props

```typescript
interface MetricTrackerCardProps {
  className?: string;  // optional class override for compact variant
}
```

### Internal State

| State | Type | Initial | Purpose |
|-------|------|---------|---------|
| Latest analytics output | `IOutputStatement \| undefined` | `undefined` | Most recent `'analytics'`-typed output from `useOutputStatements()`; component renders `null` when absent |
