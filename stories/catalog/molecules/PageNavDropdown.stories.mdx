# PageNavDropdown

A compact section-navigation dropdown used inside `CanvasPage` and `JournalPageShell`; shows the current section label and lets users jump between page anchors, using `IntersectionObserver` when uncontrolled.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `PageNavLink` | Section link type (`{ id, label, anchor }`) |
| `IntersectionObserver` | Auto-detects active section when `activeSectionId` is not provided |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `scrollToSection` | User selects a section | `sectionId: string` |

## Data Shapes

### Props

```typescript
interface PageNavDropdownProps {
  links: PageNavLink[];
  scrollToSection: (sectionId: string) => void;
  activeSectionId?: string;
}

interface PageNavLink {
  id: string;
  label: string;
}
```
