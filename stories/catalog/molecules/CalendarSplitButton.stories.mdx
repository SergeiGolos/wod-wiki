# CalendarSplitButton

A two-sided pill: the left zone triggers a primary `INavActivation` for today; the right zone opens a `CalendarCard` popover so users can choose a different date before confirming.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `INavActivation` | Primary action descriptor |
| `CalendarCard` | Date-picker popover content |
| `FIXTURE_ENTRY_DATES` | Entry dot indicators |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `primary.action.handler()` | Left zone clicked (today) | none |
| `onDateSelect` | Date chosen in calendar popover | `Date` |

## Data Shapes

### Props

```typescript
interface CalendarSplitButtonProps {
  primary: INavActivation;
  entryDates?: Date[];
  selectedDate?: Date;
  onDateSelect?: (date: Date) => void;
  size?: 'sm' | 'default';
}
```

### Internal State

| State | Type | Initial | Purpose |
|-------|------|---------|---------|
| `open` | `boolean` | `false` | Controls calendar popover visibility |
