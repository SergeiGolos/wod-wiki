# ButtonGroupDropdown

Extends `ButtonGroup` with a chevron on the right side that opens a dropdown menu of additional `INavActivation` actions; empty-label actions render as visual separators.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `INavActivation` | Action descriptor; empty-label actions become dividers |
| `src/nav/navTypes` | Type definitions |
| Radix UI Dropdown / Popover | Dropdown positioning |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `primary.action.handler()` | Primary button clicked | none |
| `actions[n].action.handler()` | Dropdown item selected | none |

## Data Shapes

### Props

```typescript
interface ButtonGroupDropdownProps {
  primary: INavActivation;
  actions: INavActivation[];  // empty label → separator
  size?: 'sm' | 'default';
  variant?: 'default' | 'primary';
}
```
