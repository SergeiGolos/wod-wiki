# CommandItem

The individual building blocks of a command-palette result list: a `CommandGroupHeading` section label, a `DefaultListItem` interactive row (with optional icon, shortcut, and subtitle), and a `CommandEmptyState` fallback.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `DefaultListItem` | Interactive list row (`src/components/list/DefaultListItem`) |
| `IListItem` / `ListItemContext` | Data and interaction types |
| `ShortcutBadge` | Keyboard shortcut display |
| Lucide icons | Optional leading icons |

## Notifies

| Callback / Event | Trigger | Payload |
|-----------------|---------|---------|
| `onSelect` (via context) | Row clicked or keyboard-activated | `IListItem` payload |

## Data Shapes

### Props

```typescript
interface DefaultListItemProps {
  item: IListItem;
  context: ListItemContext;
}

interface IListItem<T = unknown> {
  id: string;
  label: string;
  subtitle?: string;
  group?: string;
  icon?: React.ReactNode;
  shortcut?: string[];
  keywords?: string[];
  badge?: string;
  payload: T;
}

interface ListItemContext {
  isActive?: boolean;
  isSelected?: boolean;
  isDisabled?: boolean;
  onSelect?: (item: IListItem) => void;
}
```
