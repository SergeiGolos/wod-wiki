# MetricSourceRow

A runtime-block row component that displays a block's label, status, indentation depth, and inline metrics; used in timer panels, editor overlays, and analytics views.

## Depends On

| System / Token | Usage |
|----------------|-------|
| `IMetricSource` | Runtime block metrics source interface |
| `IMetric` | Individual metric data shape |
| `MetricVisualizer` | Renders the inline metric pill row |

## Notifies

None — display-only component.

## Data Shapes

### Props

```typescript
interface MetricSourceRowProps {
  label?: string;
  status?: 'pending' | 'active' | 'completed' | 'failed';
  depth?: number;         // indentation level (0, 1, 2)
  isLeaf?: boolean;       // leaf nodes render bold label
  isHeader?: boolean;     // section header styling
  metrics?: IMetric[];
  duration?: number;      // optional duration column (ms)
}
```
