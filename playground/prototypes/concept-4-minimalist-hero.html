📱 Mobile Behavior — Concept 4 (Recommended)
  • • Hero scales to 32–40px type: Headline tracks -0.06em. CTAs become full-width stacked buttons.
  • • Pillars → horizontal scroll row: 3 pillar cards become a swipeable horizontal row (120px min-width each) instead of a 3-col grid.
  • • Sandbox columns stack: Editor first, full width. HUD directly below (no border-top gap). The card wraps as one cohesive unit.
  • • HUD 3-button controls: Pause / Next / Stop retained as 3 equal columns with 48px tap targets.
  • • Run button: Full-width, inside editor section, 48px height. Triggers HUD to become visible below.
  • • Fullscreen hint: Hidden on mobile — fullscreen covers the whole device screen natively.
