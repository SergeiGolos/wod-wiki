Designer Notes — WOD-694

What problem this solves: Effort pages have YAML frontmatter with structured fields (slug, label, met, discipline, intensityTier, aliases). Editing raw YAML is error-prone. These widgets provide field-aware UI within the editor context.

Variant A (Smart Chip) is the lowest-footprint option — builds on the CodeMirror tooltip pattern already used by the autocomplete system. Best for quick slug corrections and familiar to power users.

Variant B (Side Companion) extends the existing FrontmatterCompanion component which already exists in the codebase — lowest implementation cost. Shows all fields simultaneously.

Variant C (Floating Drawer) provides the richest experience with tabs, a MET slider, and per-field UI controls. Best for infrequent but thorough edits. The bottom-sheet pattern is widely understood.

Recommendation: Variant B for immediate implementation (reuses FrontmatterCompanion, low risk). Variant C for V2 once the pattern is validated.

Implementation Handoff — WOD-695 → WOD-696

Canvas shell constraints honored: The prototype uses the real CanvasPage layout model — sticky header (data-page-sticky-boundary="true", z-30), two-column body (40% prose + 60% sticky editor via position:sticky, align-self:flex-start), and mobile reorder (single column, companion expands below editor).

Extend FrontmatterCompanion.tsx: Add effortMode: boolean prop. When true, render EffortCompanionContent sub-component. Width: 220px on desktop, full-width below editor on mobile.

Navigation invariants: ?h= section tracking must not break. No new routing model. Companion open/close state is purely local (not persisted to URL). StickyNavPanel / IntersectionObserver continues to drive active section.

Write-through: Edits in the companion must update the YAML source in real time via CodeMirror transactions. The raw YAML remains the source of truth; the companion is a structured view over it.

Exit triggers: Clicking back into prose, Escape key, explicit Discard button, or navigating away collapses the companion. Save writes + collapses.
