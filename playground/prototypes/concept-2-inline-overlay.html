📱 Mobile Behavior — Concept 2
  • • Overlay → bottom sheet: The floating right-side HUD panel becomes a bottom sheet that slides up after "Run" is tapped. Dismissible by swipe-down.
  • • Metric chips hidden: Per-line chips are hidden on mobile to preserve reading width. Only the bottom bar (movement count) is shown.
  • • Full-width editor: Editor fills 375px viewport. MacOS chrome retained for visual consistency.
  • • Syntax cards → compact list: 3-column grid collapses to horizontal list rows (icon + label + docs link).
  • • Run CTA: Prominent full-width button in bottom sheet, 48px tap target.
