📱 Mobile Behavior — Concept 1
  • • Split pane → stacked: The two equal columns collapse into a single vertical stack at <768px. Editor appears first.
  • • HUD compact strip: The full HUD collapses to: round indicator + countdown + active movement card + Run button. Next-up list is hidden by default, expandable via tap.
  • • Drag handle removed: The vertical divider disappears on mobile. Full-width editor fills the viewport.
  • • Syntax cards: 3-column grid → 1 column. Cards expand to full width.
  • • Run button: Pinned at bottom of HUD section, full width, 48px tap target.
