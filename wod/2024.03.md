# 2024 CrossFit Games - Event 3 "Firestorm"

**Category:** Competition
**Type:** For Time
**Location:** Dickies Arena, Fort Worth, Texas
**Date:** August 9, 2024 (Day 1 - Friday)

## Description

Row and burpee pull-ups with 12-minute time cap.

```wod
500m Row
30 Burpee Pullups
500m Row
```

## Breakdown

- **Modality:** Mixed cardio and gymnastics
- **Time Domain:** Short-moderate (8-12 minutes)
- **Primary Movements:** Rowing, burpee pull-up
- **Intensity:** High

## Collectible Metrics

- **Time:** Total time to complete (or reps at time cap)
