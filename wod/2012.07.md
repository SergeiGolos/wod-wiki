# 2012 CrossFit Games - Event 7 "Eva"

**Category:** CrossFit Benchmark / Competition
**Type:** For Time
**Location:** Home Depot Center, Carson, California
**Date:** July 14, 2012 (Day 4 - Saturday)

## Description

The classic benchmark "Eva" - a longer triplet for 5 rounds.

```wod
(5)
  800m Run
  30 KB Swings 70lb
  30 Pullups
```

## Scaling

| Level | Kettlebell |
|-------|------------|
| Men | 2 pood (70 lb) |
| Women | 1.5 pood (53 lb) |

- **Modality:** Mixed cardio, weightlifting, and gymnastics
- **Time Domain:** Long (25-40 minutes expected)
- **Primary Movements:** Running, kettlebell swing, pull-up
- **Intensity:** Moderate-high

## Collectible Metrics

- **Time:** Total time to complete 5 rounds
