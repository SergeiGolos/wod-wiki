# 2012 CrossFit Games - Event 9

**Category:** Competition
**Type:** For Time
**Location:** Home Depot Center, Carson, California
**Date:** July 15, 2012 (Day 5 - Sunday)

## Description

A row, push jerk, and burpee sandwich workout.

```wod
500m Row
40 Push Jerks 165lb
40 Burpees
500m Row
```

| Level | Push Jerks |
