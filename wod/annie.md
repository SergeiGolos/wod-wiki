# Annie

**Category**: CrossFit Benchmark  
**Type**: For Time  
**Difficulty**: Intermediate  

## Description
Annie is a classic CrossFit benchmark that tests gymnastics endurance and double-under skill.

```wod
(50-40-30-20-10)
  Double-Unders
  Situps
```

## Breakdown
- **Rounds**: 5 rounds with descending rep scheme (50-40-30-20-10)
- **Exercises**: 
  - Double-unders (jump rope with two passes per jump)
  - Situps
- **Goal**: Complete as fast as possible

## Target Times
- **Elite**: Under 5 minutes
- **Advanced**: 5-7 minutes  
- **Intermediate**: 7-10 minutes
- **Beginner**: 10+ minutes

## Tips
- Practice double-unders regularly to maintain efficiency
- Break up situps to avoid core fatigue
- Maintain steady pace on double-unders

## Scaling Options
- **Beginner**: Substitute single-unders (doubles: 1, singles: 3)
- **Intermediate**: Focus on consistent double-unders
- **Advanced**: Try unbroken sets when possible
