# Helen

**Category**: CrossFit Benchmark  
**Type**: For Time  
**Difficulty**: Intermediate  

## Description
Helen combines cardio with conditioning work, testing both endurance and strength.

## WOD Wiki Syntax
```
(3)
  400m Run
  21 KB Swings 53lb
  12 Pullups
```

## Breakdown
- **Rounds**: 3 rounds
- **Exercises**: 
  - 400m Run
  - Kettlebell Swings: 21 reps at 53lb (24kg)
  - Pullups: 12 reps
- **Goal**: Complete as fast as possible

## Target Times
- **Elite**: Under 8 minutes
- **Advanced**: 8-12 minutes  
- **Intermediate**: 12-15 minutes
- **Beginner**: 15+ minutes

## Tips
- Push the run but save energy for swings and pullups
- Use efficient kettlebell swing technique (hip hinge)
- Break pullups strategically to avoid failure

## Scaling Options
- **Beginner**: 300m run, 35lb KB, band-assisted pullups
- **Intermediate**: 400m run, 44lb KB, jumping pullups
- **Advanced**: Rx'd as written, consider heavier KB

## Technical Notes
- American kettlebell swings (overhead)
- Focus on hip drive, not arm lift
- Transition quickly between exercises
