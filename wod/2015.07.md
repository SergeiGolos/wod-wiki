# 2015 CrossFit Games - Event 7 "Murph"

**Category:** CrossFit Hero WOD / Competition
**Type:** For Time
**Location:** StubHub Center, Carson, California
**Date:** July 24, 2015 (Day 4 - Friday)

## Description

The classic Hero WOD "Murph" with vest.

```wod
1mile Run
100 Pullups
200 Pushups
300 Squats
1mile Run
```

- **Intensity:** Moderate-high

## Collectible Metrics

- **Time:** Total time to complete
