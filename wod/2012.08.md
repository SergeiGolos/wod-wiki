# 2012 CrossFit Games - Event 8

**Category:** Competition
**Type:** For Time
**Location:** Home Depot Center, Carson, California
**Date:** July 14, 2012 (Day 4 - Saturday)

## Description

Thrusters and chest-to-bar pull-ups in a descending rep scheme with a 12-minute time cap.

```wod
(21-18-15-12-9-6-3)
  Thrusters 95lb
  Chest to Bar Pullups
```

| Level | Thrusters |
|-------|-----------|
| Men | 95 lb |
| Women | 65 lb |

- **Modality:** Weightlifting and gymnastics
- **Time Domain:** Moderate (8-12 minutes)
- **Primary Movements:** Thruster, chest-to-bar pull-up
- **Intensity:** Very high

## Collectible Metrics
