# 2009 CrossFit Games - Event 1

**Category:** Competition
**Type:** For Time
**Location:** Aromas, California
**Date:** July 11, 2009 (Day 1 - Saturday)

## Description

A row and push jerk sandwich workout testing both cardio capacity and overhead strength.

```wod
500m Row
30 Push Jerks 135lb
500m Row
```

## Scaling

| Level | Push Jerks |
|-------|------------|
| Men | 135 lb |
| Women | 95 lb |

## Breakdown

- **Modality:** Mixed cardio and weightlifting
- **Time Domain:** Moderate (8-12 minutes expected)
- **Primary Movements:** Rowing, push jerk
- **Intensity:** High

## Collectible Metrics

- **Time:** Total time to complete
