# Jackie

**Category**: CrossFit Benchmark  
**Type**: For Time  
**Difficulty**: Intermediate  

## Description
Jackie combines cardio endurance with metabolic conditioning work.

## WOD Wiki Syntax
```
1000m Row
50 Thrusters 45lb
30 Pullups
```

## Breakdown
- **Exercises**: 
  - 1000m Row
  - Thrusters: 50 reps at 45lb (barbell)
  - Pullups: 30 reps
- **Goal**: Complete as fast as possible

## Target Times
- **Elite**: Under 8 minutes
- **Advanced**: 8-12 minutes  
- **Intermediate**: 12-15 minutes
- **Beginner**: 15+ minutes

## Tips
- Pace the row strategically (roughly 70% effort)
- Break thrusters early - 50 is a lot at any weight
- Use efficient pullup technique

## Scaling Options
- **Beginner**: 800m row, 35lb thrusters, jumping pullups
- **Intermediate**: 1000m row, 45lb thrusters, band-assisted pullups
- **Advanced**: Rx'd as written, consider 55lb thrusters

## Technical Notes
- Maintain consistent rowing pace
- Full depth on thrusters (hip crease below knee)
- Focus on fast transitions
