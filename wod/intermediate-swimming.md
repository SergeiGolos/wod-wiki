# Intermediate Swimming

**Category**: Swimming  
**Type**: Technique & Conditioning  
**Difficulty**: Intermediate  

## Description
A well-rounded swimming workout combining all competitive strokes and skills.

```wod
(4) Warmup
  50m Swim
  :30 Rest

(4)
  25m Kick
  :30 Rest

(4)
  50m Pull
  :30 Rest

(4)
  25m Swim
  :30 Rest

200m Cooldown
```

## Breakdown
- **Warmup**: 4 sets of 50m swimming
- **Kick Set**: 4 sets of 25m kicking
- **Pull Set**: 4 sets of 50m pulling (use pull buoy)
- **Main Set**: 4 sets of 25m swimming
- **Cooldown**: 200m easy swim
- **Total Distance**: 900m

## Tips
- Use pull buoy for pull sets to focus on arms
- Vary strokes during swim sets
- Maintain consistent pace throughout
- Focus on technique when fatigued

## Scaling Options
- **Intermediate**: As written
- **Advanced**: Increase distances or add intervals
- **Beginner**: Reduce distances or add more rest

## Technical Notes
- Pull sets: focus on high elbow catch
- Kick sets: use kickboard, work on flutter kick
- Swim sets: rotate through different strokes
- Cooldown: easy, focus on recovery
