# 2007 CrossFit Games - Event 2

**Category:** Competition
**Type:** For Time
**Location:** Aromas, California

Trail run event at the inaugural CrossFit Games. Athletes complete an approximately 5K trail run through the Aromas terrain.

5km Trail Run
