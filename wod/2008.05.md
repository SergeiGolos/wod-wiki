# 2008 CrossFit Games - Event 5 "Helen"

**Category:** CrossFit Benchmark / Competition
**Type:** For Time
**Location:** Aromas, California
**Date:** July 6, 2008 (Day 2 - Sunday)

## Description

The classic CrossFit benchmark workout "Helen" - a triplet performed for 3 rounds.

```wod
(3)
  400m Run
  21 KB Swings 53lb
  12 Pullups
```

## Scaling

| Level | Kettlebell |
|-------|------------|
| Men | 1.5 pood (53 lb) |
| Women | 1 pood (35 lb) |

## Breakdown

- **Modality:** Triplet of monostructural, weightlifting, and gymnastics
- **Time Domain:** Short-moderate (7-12 minutes expected)
- **Primary Movements:** Running, kettlebell swing, pull-up
- **Intensity:** High

## Collectible Metrics

- **Time:** Total time to complete 3 rounds
- Helen times are widely tracked as a CrossFit benchmark
