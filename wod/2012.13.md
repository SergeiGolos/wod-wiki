# 2012 CrossFit Games - Event 13

**Category:** Competition
**Type:** For Time
**Location:** Home Depot Center, Carson, California

## Description

The classic "Murph-style" chipper with running and bodyweight movements.


- **Intensity:** Moderate-high

## Collectible Metrics
