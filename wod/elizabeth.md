# Elizabeth

**Category**: CrossFit Benchmark  
**Type**: For Time  
**Difficulty**: Intermediate  

## Description
Elizabeth tests Olympic lifting technique combined with upper body gymnastics strength.

## WOD Wiki Syntax
```wod
(21-15-9)
  Clean 135lb
  Ring Dips
```

## Breakdown
- **Rounds**: 3 rounds with descending rep scheme (21-15-9)
- **Exercises**: 
  - Power Cleans at 135lb
  - Ring Dips
- **Goal**: Complete as fast as possible

## Target Times
- **Elite**: Under 4 minutes
- **Advanced**: 4-7 minutes  
- **Intermediate**: 7-10 minutes
- **Beginner**: 10+ minutes

## Tips
- Focus on clean technique - squat clean recommended
- Use efficient ring dip technique (false grip)
- Break sets strategically to avoid failure

## Scaling Options
- **Beginner**: Reduce clean weight to 95lb, use box/bench dips
- **Intermediate**: 115lb cleans, band-assisted ring dips
- **Advanced**: Rx'd as written, consider 155lb cleans

## Technical Notes
- Squat cleans are more efficient than power cleans
- Maintain ring turn-out for shoulder health
- Use hips for clean power, not just arms
