# Olympic Distance Endurance

**Type**: Aerobic Endurance  
**Duration**: 60-75 minutes  
**Difficulty**: Intermediate  

### Description
This workout builds the sustained aerobic capacity needed for Olympic-distance triathlon swims (1.5km). The session uses extended intervals at race pace with minimal rest to develop the metabolic efficiency triathletes need to swim strongly while conserving energy for the bike and run. Open water simulation elements prepare athletes for the rhythm changes and sighting demands of race day.

```wod
(3) Warmup
  150m Freestyle
  :30 Rest

(6) Sighting Integration
  50m Sighting Swim
  :20 Rest

(10) Steady State
  100m Freestyle
  :15 Rest

(4) Tempo Variation
  + 75m Moderate
  + :10 Rest
  + 75m Race Pace
  + :20 Rest

(6) Negative Split 100s
  100m Freestyle
  :20 Rest

(4) Draft Practice
  100m Draft Position
  :20 Rest

200m Cooldown
```

### Breakdown
- **Warm-up**: 450m progressive effort
- **Sighting Integration**: 300m practicing sighting rhythm
- **Steady State**: 1,000m at sustainable race pace
- **Tempo Variation**: 600m alternating moderate and race pace
- **Negative Split 100s**: 600m with second 50m faster
- **Draft Practice**: 400m positioning work with lane partners
- **Cooldown**: 200m easy swimming

### Collectible Metrics
- Pace consistency across steady state set
- Heart rate at race pace
- Sighting accuracy percentage
