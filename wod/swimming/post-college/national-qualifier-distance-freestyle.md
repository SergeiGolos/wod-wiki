# National Qualifier Distance Freestyle

**Type**: Elite Endurance  
**Duration**: 100-110 minutes  
**Difficulty**: Elite  

### Description
This workout targets post-collegiate swimmers pursuing national qualifying times in distance freestyle events (400m, 800m, 1500m). The session combines high-volume aerobic development with race-specific threshold work that builds the sustained speed needed for elite-level competition. Swimmers should maintain precise pace control throughout the main sets, training their internal clocks to recognize goal race pace without constant verification.

The workout assumes access to a 50m long course pool during summer months, though it can be adapted to short course yards during winter training. Athletes training alone should use a pace clock or waterproof timing device to verify interval holds. Those with training partners should rotate lead positions to practice drafting and varied pacing.

```wod
(4) Warmup
  200m Freestyle
  :30 Rest

(4) Pre-Set Activation
  100m IM
  :20 Rest
  50m Build
  :20 Rest

(3) Threshold Ladder
  + 500m Freestyle
  + :45 Rest
  + 400m Freestyle
  + :30 Rest
  + 300m Freestyle
  + :20 Rest
  + 200m Freestyle
  + :15 Rest
  + 100m Freestyle
  + 1:00 Rest

(8) Pace 200s
  200m Freestyle
  :30 Rest

(6) Kick Endurance
  100m Kickboard
  :20 Rest

(4) Pull Threshold
  150m Freestyle Pull
  :30 Rest

300m Cooldown
```

### Breakdown
- **Warm-up**: 800m progressive effort with technique focus
- **Pre-Set Activation**: 600m IM work activating all strokes
- **Threshold Ladder**: 1,500m descending distance at threshold pace
- **Pace 200s**: 1,600m at goal 400m/800m race pace
- **Kick Endurance**: 600m sustained leg output
- **Pull Threshold**: 600m upper body threshold work
- **Cooldown**: 300m easy swimming

### Collectible Metrics
- Average pace on threshold ladder (should descend with distance)
- Pace consistency on 200s (goal: within 2 seconds across all 8)
- Stroke count stability under fatigue
- Heart rate recovery at set completion
