# Mid-Distance Championship Preparation

**Type**: Race-Specific  
**Duration**: 105 minutes  
**Difficulty**: Elite  

### Description
This championship-caliber workout prepares mid-distance swimmers (200m and 400m events) for the specific demands of elite racing. The session uses pace training at goal race speeds, broken swims that simulate race execution, and negative split work that develops finishing speed. This workout is designed for the taper phase when swimmers are reducing volume while maintaining race-specific intensity and sharpening their feel for pace.

```wod
(3) Warmup
  200m Freestyle
  :30 Rest

(4) Pace Development
  100m Race Pace
  1:00 Rest

(8) Broken 200 Prep
  + 50m Race Pace
  + :15 Rest
  + 50m Race Pace
  + :15 Rest
  + 50m Race Pace
  + :15 Rest
  + 50m Race Pace
  + 3:00 Rest

(4) Negative Split 150s
  150m Freestyle
  1:30 Rest

(6) Pace 100s Descending
  100m Freestyle
  :45 Rest

(4) Lactate Tolerance
  75m Max Effort
  2:00 Rest

250m Cooldown
```

### Breakdown
- **Warm-up**: 600m easy swimming with progressive build
- **Pace Development**: 400m establishing goal race pace
- **Broken 200 Prep**: 800m total in 4 rounds, simulating 200m race
- **Negative Split 150s**: 600m practicing back-half speed
- **Pace 100s Descending**: 600m with decreasing rest
- **Lactate Tolerance**: 300m maximum effort, racing fatigue
- **Cooldown**: 250m easy recovery

### Collectible Metrics
- Actual pace vs. goal pace on all race-pace efforts
- Negative split differential on 150s
- Stroke efficiency at race pace
- Lactate clearance rate between sets
