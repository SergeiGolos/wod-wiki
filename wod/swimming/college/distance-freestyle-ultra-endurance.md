# Distance Freestyle Ultra-Endurance

**Type**: Aerobic Endurance  
**Duration**: 120 minutes  
**Difficulty**: Advanced-Elite  

### Description
This high-volume distance workout builds the massive aerobic base required for 500m, 1000m, and 1650m freestyle events. The main set uses a ladder structure that progressively challenges swimmers with increasing distances while maintaining prescribed pace times. This session is typically performed during mid-season training phases when building peak endurance capacity. Swimmers should maintain stroke efficiency throughout, with coaches monitoring distance per stroke and stroke rate metrics.

```wod
(4) Warmup
  200m Freestyle
  :30 Rest

(4) Pre-Set
  150m IM
  :20 Rest

(3) Ladder Main Set
  + 400m Freestyle
  + :45 Rest
  + 300m Freestyle
  + :30 Rest
  + 200m Freestyle
  + :20 Rest
  + 100m Freestyle
  + 1:00 Rest

(2) Ladder Repeat
  + 400m Freestyle
  + :45 Rest
  + 300m Freestyle
  + :30 Rest
  + 200m Freestyle
  + :20 Rest
  + 100m Freestyle
  + 1:00 Rest

(8) Kick Set
  100m Kickboard
  :20 Rest

(6) Pull Set
  100m Freestyle Pull
  :20 Rest

300m Cooldown
```

### Breakdown
- **Warm-up**: 800m progressive effort preparing all energy systems
- **Pre-Set**: 600m IM rotation activating all muscle groups
- **Ladder Main Set**: 2,000m first progression (400→300→200→100)
- **Ladder Repeat**: 2,000m second progression at same or faster pace
- **Kick Set**: 800m sustained leg output
- **Pull Set**: 600m upper body endurance focus
- **Cooldown**: 300m easy swimming for recovery

### Collectible Metrics
- Pace hold percentage across ladder sets
- Stroke count consistency throughout main set
- Heart rate recovery between sets
