# General Fitness Session

**Type**: Balanced Fitness  
**Duration**: 60-75 minutes  
**Difficulty**: Beginner-Intermediate  

### Description
This balanced workout serves as an ideal entry point for masters swimmers seeking general fitness benefits without competitive aspirations. The session combines technique work with moderate-intensity swimming to improve cardiovascular health and muscular endurance. Swimmers can adjust distances and rest intervals based on their current fitness level, making this workout accessible to a wide range of abilities within a lane.

```wod
(3) Warmup
  100m Freestyle
  :30 Rest

(4) Technique Focus
  50m Drill
  :20 Rest
  50m Swim
  :20 Rest

(6) Main Set
  75m Freestyle
  :30 Rest

(4) Kick Set
  50m Kickboard
  :30 Rest

(4) Pull Set
  75m Freestyle Pull
  :30 Rest

(6) Cardio Build
  50m Freestyle
  :20 Rest

150m Cooldown
```

### Breakdown
- **Warm-up**: 300m easy swimming to prepare body
- **Technique Focus**: 400m alternating drills and swimming
- **Main Set**: 450m moderate effort with consistent rest
- **Kick Set**: 200m leg work for body position
- **Pull Set**: 300m upper body focus
- **Cardio Build**: 300m building heart rate
- **Cooldown**: 150m easy swimming

### Collectible Metrics
- Total distance completed
- Perceived effort rating (1-10)
- Technique focus points practiced
