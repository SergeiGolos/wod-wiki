# IM Training Session

**Type**: Individual Medley  
**Duration**: 90 minutes  
**Difficulty**: Advanced  

### Description
This comprehensive IM workout develops proficiency across all four competitive strokes while building the specific endurance needed for 200m and 400m IM events. The session includes stroke-specific sets for each discipline plus transition work between strokes. The main set challenges swimmers to maintain stroke integrity while fatigued, preparing them for the demanding final freestyle leg of IM races.

```wod
(3) Warmup
  150m IM
  :30 Rest

(4) Butterfly Focus
  50m Butterfly
  :30 Rest

(4) Backstroke Focus
  50m Backstroke
  :30 Rest

(4) Breaststroke Focus
  50m Breaststroke
  :30 Rest

(4) Freestyle Focus
  50m Freestyle
  :30 Rest

(6) IM Main Set
  100m IM
  :45 Rest

(4) Transition Work
  75m Stroke Transitions
  :30 Rest

200m Cooldown
```

### Breakdown
- **Warm-up**: 450m IM continuous for full body activation
- **Butterfly Focus**: 200m building stroke rhythm and timing
- **Backstroke Focus**: 200m emphasizing rotation and catch
- **Breaststroke Focus**: 200m working on timing and glide
- **Freestyle Focus**: 200m at moderate effort
- **IM Main Set**: 600m building race-pace endurance
- **Transition Work**: 300m practicing smooth stroke changes
- **Cooldown**: 200m easy swimming

### Collectible Metrics
- Split times for each stroke segment
- Stroke count per 50m for each stroke
- Transition efficiency rating
