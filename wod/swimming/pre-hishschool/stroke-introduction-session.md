# Stroke Introduction Session

**Type**: Technique Focus  
**Duration**: 45 minutes  
**Difficulty**: Beginner  

### Description
This introductory session focuses on helping new swimmers become comfortable with freestyle mechanics. The workout emphasizes body position, breathing timing, and arm recovery. Coaches should position themselves at poolside to provide immediate feedback on technique elements. This session works well as the first structured practice for swimmers who can already perform basic water safety skills.

```wod
(4) Warmup
  25m Freestyle
  :30 Rest

(6) Drill Focus
  25m Catch-Up Drill
  :20 Rest

(4) Kick Focus
  25m Kickboard Kick
  :30 Rest

100m Cooldown
```

### Breakdown
- **Warm-up**: 100m total - establishes comfort and prepares muscles
- **Drill Focus**: 150m total - catch-up drill teaches proper arm timing and extension
- **Kick Focus**: 100m total - builds leg strength and body position awareness
- **Cooldown**: 100m easy swimming to reinforce relaxed technique

### Collectible Metrics
- Time per 25m segment (to track improvement over weeks)
- Number of breaths per length (efficiency indicator)
