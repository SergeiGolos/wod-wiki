# Endurance Building

**Type**: Endurance  
**Duration**: 55 minutes  
**Difficulty**: Beginner-Intermediate  

### Description
This workout progressively builds aerobic capacity for young swimmers while maintaining focus on technique. The main set uses descending rest intervals to challenge swimmers to maintain form under mild fatigue. Coaches should monitor stroke quality and encourage swimmers to focus on efficiency rather than speed during longer distances.

```wod
(2) Warmup
  100m Freestyle
  :45 Rest

(8) Main Set
  50m Freestyle
  :30 Rest

(4) Kick Set
  50m Kickboard
  :30 Rest

(4) Pull Set
  50m Freestyle Pull
  :30 Rest

150m Cooldown
```

### Breakdown
- **Warm-up**: 200m total at easy pace, focus on relaxed breathing
- **Main Set**: 400m total with consistent rest - builds aerobic base
- **Kick Set**: 200m total - develops leg endurance and body position
- **Pull Set**: 200m total - isolates upper body and builds arm strength
- **Cooldown**: 150m easy swimming with mixed strokes

### Collectible Metrics
- Average time per 50m in main set
- Perceived effort rating (1-10) at completion
