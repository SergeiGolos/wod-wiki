# Beginner Friendly Swimming

**Category**: Swimming  
**Type**: Technique & Conditioning  
**Difficulty**: Beginner  

## Description
A swimming workout focused on technique development and basic conditioning for new swimmers.

```wod
(6) Warmup
  25m Swim
  :20 Rest

100m Kick

(6) Warmup
  25m Swim
  :20 Rest

100m Kick

(6) Warmup
  25m Swim
  :20 Rest
  
100m Cooldown
```

## Breakdown
- **Warmup Sets**: 3 sets of 25m swimming with :20 rest
- **Kick Sets**: 2 sets of 100m kicking
- **Cooldown**: 100m easy swim
- **Total Distance**: 550m
- **Focus**: Technique and comfort in water

## Tips
- Focus on breathing technique during warmup
- Use kickboard for kick sets
- Slow, controlled movements
- Rest as needed

## Scaling Options
- **Beginner**: As written, focus on form
- **Intermediate**: Add more sets or increase distance
- **Advanced**: Reduce rest, add intervals

## Technical Notes
- Focus on proper stroke mechanics
- Breathing every 2-3 strokes
- Keep kicks small and efficient
- Use fins if needed for technique work
