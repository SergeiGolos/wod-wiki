# Long Distance Swimming

**Category**: Swimming  
**Type**: Endurance Training  
**Difficulty**: Advanced  

## Description
A distance-focused swimming workout building endurance for longer events.

## WOD Wiki Syntax
```
(3) Warmup
  200m Swim
  :30 Rest

(6)
  100m Kick
  :30 Rest

(6)
  100m Pull
  :30 Rest

(6)
  100m Swim
  :30 Rest

500m Cooldown
```

## Breakdown
- **Warmup**: 3 sets of 200m swimming
- **Kick Set**: 6 sets of 100m kicking
- **Pull Set**: 6 sets of 100m pulling
- **Main Set**: 6 sets of 100m swimming
- **Cooldown**: 500m easy swim
- **Total Distance**: 2300m

## Tips
- Focus on steady pacing for endurance
- Use pull buoy to conserve energy during pull sets
- Practice negative splitting (second half faster)
- Stay hydrated and fueled

## Scaling Options
- **Advanced**: As written
- **Elite**: Increase distances or reduce rest
- **Intermediate**: Reduce sets to 4-5 each

## Technical Notes
- Focus on efficient stroke technique
- Use pacing strategies (even splits)
- Practice breathing patterns for endurance
- Good for triathlon training
