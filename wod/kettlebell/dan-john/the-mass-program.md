## Workout 5: The Mass Program

**Category**: Dan John Hypertrophy  
**Type**: High Volume Strength  
**Difficulty**: Intermediate to Advanced

### Description

The Mass Program is Dan John's approach to building muscle with kettlebells, combining the Armor Building Complex with strategic volume progression. The program uses a wave loading pattern, increasing volume over several weeks before deloading and repeating with heavier weights.

The program emphasizes compound movements and high tension, creating the metabolic and mechanical stress necessary for hypertrophy while maintaining the functional strength and movement quality that defines Dan John's approach. The combination of pressing, squatting, and pulling movements develops balanced musculature.

### Week 1-2

```wod
(5)
  + 2 Double Clean 20kg
  + 1 Double Press 20kg
  + 3 Double Front Squat 20kg
  :60 Rest
```
### Week 3-4

```wod
(5)
  + 2 Double Clean 24kg
  + 1 Double Press 24kg
  + 3 Double Front Squat 24kg
  :60 Rest
```
### Week 5-6

```wod
(5)
  + 2 Double Clean 24kg
  + 2 Double Press 24kg
  + 3 Double Front Squat 24kg
  :60 Rest
```
