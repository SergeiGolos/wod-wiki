## Workout 5: 2 Kettlebells 12 Weeks Program

**Category**: Geoff Neupert Progressive Program  
**Type**: Progressive Strength Complex  
**Difficulty**: Beginner to Advanced

### Description

This year-long program takes practitioners from beginner through advanced double kettlebell complexes across 12 weeks. The program uses 15-minute sessions, 5 days per week, with a focus on progressive complexity and load. Each week introduces new elements while building on previous foundations.

The program is designed for those committed to long-term development, with each phase building on the previous one. Starting with basic double movements, the program progresses to complex sequences that challenge strength, coordination, and conditioning simultaneously.

### Week 1-4 Foundation

```wod
(5)
  3 Double Clean 16kg
  :30 Rest
```
### Week 5-8 Development

```wod
(5)
  + 2 Double Clean 20kg
  + 2 Double Press 20kg
  :45 Rest
```
### Week 9-12 Advanced

```wod
(5)
  + 2 Double Clean 24kg
  + 2 Double Press 24kg
  + 3 Double Front Squat 24kg
  :60 Rest
```


### Breakdown

- **Rounds/Duration**: 12 weeks, 15 minutes per session
- **Exercises**: Progressive double kettlebell movements
- **Goal**: Mastery of double kettlebell training

### Collectible Metrics

- Weight progression
- Complex completion
- Session consistency
