## Workout 1: Biathlon (Jerk + Snatch)

**Category**: Girevoy Sport Competition  
**Type**: Competition Event  
**Difficulty**: Advanced

### Description

Biathlon is a two-event competition consisting of the Jerk followed by the Snatch. In competition, athletes perform a 10-minute jerk set with two kettlebells, rest for at least 30 minutes, then perform a 10-minute snatch set with one kettlebell. The total reps from both events determine the winner.

This workout simulates competition conditions, though training sets may be shorter than 10 minutes as athletes build capacity. The focus is on developing efficiency, pace, and the specific endurance required for each lift.

```wod
Biathlon Training:
10:00 ? Double KB Jerk 24kg

30:00 Rest

10:00 ? KB Snatch 24kg
```

### Breakdown

- **Rounds/Duration**: 10 minutes per event
- **Exercises**: Double Kettlebell Jerk, Kettlebell Snatch
- **Goal**: Maximum reps in each 10-minute set

### Competition Standards

| Weight Class | Jerk Weight | Snatch Weight | Time |
|--------------|-------------|---------------|------|
| Men Amateur | 2x16kg | 16kg | 10:00 |
| Men Professional | 2x24kg | 24kg | 10:00 |
| Men Elite | 2x32kg | 32kg | 10:00 |
| Women Amateur | 16kg | 12kg | 10:00 |
| Women Professional | 20kg | 16kg | 10:00 |

### Collectible Metrics

- Jerk reps per set
- Snatch reps per set
- Pace (reps per minute)
