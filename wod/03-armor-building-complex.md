# Armor Building Complex (ABC)

**Category**: Dan John  
**Type**: EMOM / Complex  
**Difficulty**: Intermediate to Advanced

## Description

Dan John's legendary complex designed to build "armor" - functional muscle and work capacity. The complex consists of 2 Cleans + 1 Press + 3 Front Squats performed without setting the bells down. Can be done as EMOM or AMRAP.

---

## Session 1: ABC EMOM (Standard)

The classic EMOM format - one round per minute.

```wod
(15) 1:00 EMOM
  + 2 Double KB Clean ?kg
  + 1 Double KB Press ?kg
  + 3 Double KB Front Squat ?kg
```

## Collectible Metrics

- **Rounds completed**: Number of EMOM rounds finished
- **Weight**: Bell weight used (commonly 2x24kg)
- **Quality**: Did you complete each round within the minute?

---

## Session 2: ABC AMRAP

As many rounds as possible in a fixed time window.

```wod
15:00 AMRAP
  + 2 Double KB Clean ?kg
  + 1 Double KB Press ?kg
  + 3 Double KB Front Squat ?kg
```

## Collectible Metrics

- **Total rounds**: Count of complete ABC rounds
- **Total reps**: 60 cleans, 30 presses, 90 squats at 30 rounds
- **Density**: Rounds per minute average

---

## Session 3: Single Bell ABC

For those with only one kettlebell - unilateral work.

```wod
(20) 1:00 EMOM
  + 2 KB Clean Left ?kg
  + 1 KB Press Left ?kg
  + 2 KB Squat Left ?kg
  + 2 KB Clean Right ?kg
  + 1 KB Press Right ?kg
  + 2 KB Squat Right ?kg
```

## Collectible Metrics

- **Rounds completed**: Each round works both sides
- **Weight**: Single bell weight
- **Asymmetry check**: Compare left vs right quality

## Session 6: ABC Ladder

Progressive round counts building volume.

```wod
(1-2-3-4-5)
  - 1 ABC Round ?kg
  :60 Rest
  - 2 ABC Rounds ?kg
  :60 Rest
  - 3 ABC Rounds ?kg
  :60 Rest
  - 4 ABC Rounds ?kg
  :60 Rest
  - 5 ABC Rounds ?kg
```

### Alternative Syntax:

```wod
(1-2-3-4-5)
  + 2 Double KB Clean ?kg
  + 1 Double KB Press ?kg
  + 3 Double KB Front Squat ?kg
```

## Collectible Metrics

- **Total rounds**: 15 rounds (1+2+3+4+5)
- **Progressive fatigue**: How well do you maintain form?
- **Rest quality**: Recovery between ladder rungs
