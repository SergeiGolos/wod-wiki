# Syntax: Agentic Skill

WOD Wiki is designed to be "LLM-First." We provide a specialized skill (`SKILL.md`) that teaches Large Language Models how to perfectly encode workouts into this syntax.

## Why Agentic?
- **Natural Language to Code**: Send a photo of a whiteboard or a text description to an LLM, and get back executable WOD Wiki syntax.
- **Structured Extraction**: LLMs can identify movements, reps, weights, and complex loop structures that regular parsers might miss.

## Using the Skill
You can use our `SKILL.md` with:
- Custom GPTs (OpenAI)
- Claude Projects (Anthropic)
- Gemini Gems (Google)
- Local LLMs (Ollama)
