# Syntax: Repeaters

Repeaters allow you to define varying rep counts or sets for a group of exercises.

## Rep Schemes
Use parentheses `()` with dashes to define reps per round.
```wod
(21-15-9)
  Thrusters 95lb
  Pullups
```
This creates 3 rounds:
1. 21 reps of each
2. 15 reps of each
3. 9 reps of each

## Simple Sets
You can also specify a simple number of sets.
```wod
(5 Sets)
  10 Deadlift 225lb
  1:00 Rest
```
