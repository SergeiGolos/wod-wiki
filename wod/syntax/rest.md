# Syntax: Rest Periods

Rest periods can be explicit or built into intervals.

Explicit rest between sets:

```wod
5 Rounds
  Deadlift 315lb
  2:00 Rest
```

Implicit rest within an AMRAP:

```wod
20:00
  (AMRAP)
    10 Pushups
    10 Situps
    10 Squats
    1:00 Rest
```
