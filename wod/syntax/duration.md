# Syntax: Duration (For Time)

The simplest workout type. The timer counts **up** and the workout ends when you finish all exercises.

```wod
100 Burpees
```

Or a classic "For Time" workout:

```wod
# Fran
(21-15-9)
  Thrusters 95lb
  Pullups
```
