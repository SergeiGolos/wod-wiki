# Nancy

**Category**: CrossFit Benchmark  
**Type**: For Time  
**Difficulty**: Intermediate  

## Description
Nancy combines running endurance with Olympic lifting skill under fatigue.

```wod
(5)
  400m Run
  15 Overhead Squats 95lb
```

## Breakdown
- **Rounds**: 5 rounds
- **Exercises**: 
  - 400m Run
  - Overhead Squats: 15 reps at 95lb
- **Goal**: Complete as fast as possible

## Target Times
- **Elite**: Under 12 minutes
- **Advanced**: 12-18 minutes  
- **Intermediate**: 18-25 minutes
- **Beginner**: 25+ minutes

## Tips
- Pace the runs - they add up quickly
- Focus on overhead squat technique when tired
- Break OHS sets strategically to avoid failure

## Scaling Options
- **Beginner**: 300m run, 65lb overhead squats
- **Intermediate**: 400m run, 75lb overhead squats
- **Advanced**: Rx'd as written, consider 115lb

## Technical Notes
- Full depth on overhead squats (hip crease below knee)
- Active shoulders throughout the movement
- Use hook grip if needed for bar control
