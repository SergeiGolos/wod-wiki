# 2011 CrossFit Games - Event 5

**Category:** Competition
**Type:** Max Reps
**Location:** Home Depot Center, Carson, California
**Date:** July 30, 2011 (Day 2 - Saturday)

## Description

Two separate max rep tests - handstand push-ups followed by pistols.

```wod
3:00 ? Handstand Pushups

3:00 ? Pistols
```

## Breakdown

- **Modality:** Gymnastics (bodyweight strength)
- **Time Domain:** Two 3-minute windows
- **Primary Movements:** Handstand push-up, pistol (single-leg squat)
- **Intensity:** High
- **Skill Level:** Intermediate-Advanced

## Collectible Metrics

- **Reps:** Total handstand push-ups in 3 minutes
- **Reps:** Total pistols in 3 minutes
- The `?` makes reps the collectible for each movement
