# 2007 CrossFit Games - Event 1

**Category:** Competition
**Type:** For Time
**Location:** Aromas, California
**Date:** June 30, 2007 (Day 1 - Saturday)

## Description

The first event of the inaugural CrossFit Games. Athletes begin with a 1000 meter row, followed by a classic CrossFit couplet performed for 5 rounds.

```wod
1000m Row

(5)
  25 Pullups
  7 Push Jerks 135lb
```

## Scaling

| Level | Push Jerks |
|-------|------------|
| Men | 135 lb |
| Women | 95 lb |

## Breakdown

- **Modality:** Mixed cardio and weightlifting
- **Time Domain:** Moderate (8-15 minutes expected)
- **Primary Movements:** Rowing, pulling strength, overhead pressing

## Collectible Metrics

- **Time:** Total time to complete all work
- This is a "for time" workout where the athlete is prescribed the work and discovers how long it takes
