## Workout 6: Easy Muscle

**Category**: Geoff Neupert Hypertrophy  
**Type**: Progressive Hypertrophy  
**Difficulty**: Intermediate

### Description

Easy Muscle is Geoff Neupert's hypertrophy-focused program designed specifically for men over 40, though effective for all ages. The program builds muscle through strategic volume and tension, using sub-maximal weights and frequent practice rather than training to failure.

The program uses a "reverse ladder" structure, starting with higher reps and progressively decreasing as fatigue sets in, allowing for maximum total volume without excessive fatigue. This approach builds muscle while respecting recovery capacity, making it sustainable for long-term practice.

```wod
(3)
  10 Double Clean 20kg
  :30 Rest
  8 Double Press 20kg
  :30 Rest
  6 Double Front Squat 20kg
  :30 Rest
  4 Double Clean 20kg
  :30 Rest
  2 Double Press 20kg
  :60 Rest
```

### Breakdown

- **Rounds/Duration**: 3 rounds
- **Exercises**: Double Clean, Double Press, Double Front Squat
- **Goal**: Build muscle with sustainable training

### Collectible Metrics

- Weight used
- Total volume
- Recovery quality
