## Workout 5: The Finisher (Metabolic Conditioning)

**Category**: Keith Weber Metabolic  
**Type**: High-Intensity Finisher  
**Difficulty**: Intermediate to Advanced

### Description

The Finisher is designed to be performed at the end of a training session to maximize metabolic output and test mental toughness. The short, intense format uses high-repetition swings and squats to create maximum fatigue in minimum time.

The workout can be used as a standalone conditioning session or appended to a strength workout for additional conditioning stimulus. The simplicity of the movements allows for maximum intensity with minimal technical degradation under fatigue.

### The Finisher

```wod
(5)
  20 Two-Hand KB Swing 20kg
  10 Goblet Squat 20kg
  :30 Rest
```
### Tabata Finish

```wod
(8)
  :20 KB Swing 16kg
  :10 Rest
```


### Breakdown

- **Rounds/Duration**: 8-12 minutes
- **Exercises**: Two-Hand Swing, Goblet Squat
- **Goal**: Maximum intensity in minimum time

### Collectible Metrics

- Total reps completed
- Time to complete
- Heart rate response
