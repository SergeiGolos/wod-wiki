# Pro Circuit Race Week Preparation

**Type**: Competition Sharpening  
**Duration**: 60-70 minutes  
**Difficulty**: Elite  

### Description
This pre-competition workout prepares post-collegiate swimmers for professional meet formats including the ISL, World Cup, and Grand Prix events. The session primes the neuromuscular system for maximum performance while maintaining freshness for racing. Athletes should complete this session 24-48 hours before competition, adjusting intensity based on travel fatigue and prelims/finals scheduling.

The workout emphasizes race-pace rehearsal with generous recovery, ensuring athletes feel sharp and confident. Swimmers should visualize their races during rest intervals, rehearsing tactical decisions and key technical focus points. This mental preparation is essential for professional competition where margins between advancement and elimination are often hundredths of seconds.

```wod
(3) Warmup
  150m Freestyle
  :30 Rest

(4) Activation Build
  50m Build
  :30 Rest

(3) Race Pace Primary
  100m Race Pace
  2:00 Rest

(4) Sprint Primers
  25m Sprint
  1:30 Rest

(2) Race Pace Secondary
  50m Race Pace
  1:30 Rest

(4) Start Rehearsal
  15m Dive
  1:00 Rest

(3) Pace Confirmation
  50m Goal Pace
  1:00 Rest

150m Cooldown
```

### Breakdown
- **Warm-up**: 450m easy progressive swimming
- **Activation Build**: 200m building to race pace
- **Race Pace Primary**: 300m at goal pace for primary event
- **Sprint Primers**: 100m maximum effort neural activation
- **Race Pace Secondary**: 100m at pace for secondary events
- **Start Rehearsal**: 60m dive work
- **Pace Confirmation**: 150m confirming race pace feel
- **Cooldown**: 150m easy swimming

### Collectible Metrics
- Actual times vs. goal times on all race-pace efforts
- Stroke rate consistency
- Perceived sharpness rating (1-10)
- Technical execution confidence level
