# Collegiate IM Championship

**Type**: Individual Medley  
**Duration**: 110 minutes  
**Difficulty**: Elite  

### Description
This elite IM workout develops the comprehensive skills needed for 200m and 400m IM success at the NCAA level. The session includes dedicated work on each stroke, transition efficiency between strokes, and race-pace rehearsal for full IM distances. Advanced swimmers will focus on maintaining stroke integrity through fatigue while developing the tactical awareness to manage energy distribution across all four strokes.

```wod
(3) Warmup
  200m IM
  :30 Rest

(6) Butterfly Strength
  50m Butterfly
  :30 Rest

(6) Backstroke Speed
  50m Backstroke
  :30 Rest

(6) Breaststroke Power
  50m Breaststroke
  :30 Rest

(6) Freestyle Anchor
  50m Freestyle
  :30 Rest

(6) IM Pace Set
  100m IM
  :45 Rest

(4) Broken 400 IM
  + 100m Butterfly
  + :30 Rest
  + 100m Backstroke
  + :30 Rest
  + 100m Breaststroke
  + :30 Rest
  + 100m Freestyle
  + 4:00 Rest

(6) Transition Focus
  75m IM Transitions
  :30 Rest

300m Cooldown
```

### Breakdown
- **Warm-up**: 600m IM continuous for neuromuscular activation
- **Butterfly Strength**: 300m building stroke power and rhythm
- **Backstroke Speed**: 300m rotation and tempo work
- **Breaststroke Power**: 300m timing and propulsion focus
- **Freestyle Anchor**: 300m finishing speed development
- **IM Pace Set**: 600m building race-specific endurance
- **Broken 400 IM**: 1,600m total in 4 rounds, full IM rehearsal
- **Transition Focus**: 450m smooth stroke changes
- **Cooldown**: 300m easy swimming

### Collectible Metrics
- Stroke-specific splits within IM sets
- Transition time between strokes
- Stroke count consistency across all four strokes
- Heart rate recovery patterns
