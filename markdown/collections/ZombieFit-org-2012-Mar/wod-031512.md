## Warm Up

Run 1/4 mile OR row 250m OR bike 1/2 mile.

3 rounds of:
- 25m quadrupedal movement
- 10 Squats
- 10 Sit Ups
- 10 Double Unders (or 50 Single Unders)
- 10 Push Ups

## WOD

Hold maximal distance possible on each interval.

### Étudiant — Beginner

```wod
(5)
  1:00 Run / Swim / Bike / Row ?m
  :30 Rest
```
