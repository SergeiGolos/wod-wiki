## Warm Up

Run 1/4 mile | row 250m | bike 2 miles.

```wod
(3)
  25m Quadrupedal Movement
  10 Push Ups
  10 Sit Ups
  10 Pull Ups
  10 Air Squats
```

## WOD

### Étudiant — Beginner

```wod
(15-7-9)
  12 inch Box Jumps
  Push Ups
  Air Squats
  Jumping Pull Ups
```

### Avancée — Advanced

```wod
(21-15-9)
  24 inch Box Jumps
  Push Ups
  Air Squats
  Pull Ups
```

### Traceur — Expert

```wod
(21-15-9)
  30 inch Box Jumps
  Push Ups
  Air Squats
  Pull Ups
```

If needed, substitute box jumps with tuck jumps.
