## Warm Up

Run 1/4 mile | row 250m | bike 2 miles.

```wod
(3)
  10 Push Ups
  10 Sit Ups
  10 Pull Ups
  10 Air Squats
```

## WOD

### Étudiant — Beginner

Perform single-unders with the following rep scheme. If you do not have a jump rope, sub with tuck jumps in rounds of 30-20-10.

```wod
(100-80-60-40-20)
  Single Unders
  Sit Ups
```

### Avancée — Advanced

WOD as RX'd. If you do not have a jump rope, sub double unders with tuck jumps.

```wod
(50-40-30-20-10)
  Double Unders
  Sit Ups
```

### Traceur — Expert

RX version — full Annie rep scheme. If needed, substitute tuck jumps for double unders.

```wod
(50-40-30-20-10)
  Double Unders
  Sit Ups
```
