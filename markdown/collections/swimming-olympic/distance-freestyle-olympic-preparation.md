# Distance Freestyle Olympic Preparation

**Type**: Elite Distance  
**Duration**: 120-140 minutes  
**Difficulty**: Olympic  

### Description
This world-class distance workout prepares athletes for Olympic finals in the 400m, 800m, and 1500m freestyle events. The session develops the extraordinary aerobic capacity and tactical awareness required for distance racing at the highest international level. The main set challenges athletes to maintain world-record pace under severe metabolic stress, building the physiological machinery and mental fortitude necessary for Olympic podium performances.

Athletes should complete this session during the quality phase of training, approximately 8-12 weeks before major competition. The prescribed paces should match or exceed goal Olympic pace, with athletes training their bodies and minds to sustain speeds that will challenge for medals. The set structure allows coaches to monitor fatigue accumulation and adjust volume based on individual response.

```wod
(4) Warmup
  200m Freestyle
  :30 Rest

(4) Pre-Set
  150m IM
  :20 Rest
  100m Build
  :30 Rest

(3) Threshold Development
  + 600m Freestyle
  + :45 Rest
  + 500m Freestyle
  + :30 Rest
  + 400m Freestyle
  + :20 Rest
  + 300m Freestyle
  + :15 Rest
  + 200m Freestyle
  + 1:00 Rest

(6) Race Pace 300s
  300m Freestyle
  :45 Rest

(8) Negative Split 200s
  200m Freestyle
  :30 Rest

(8) Kick Race Pace
  100m Kick
  :20 Rest

(4) Pull Power
  200m Freestyle Pull
  :30 Rest

400m Cooldown
```

### Breakdown
- **Warm-up**: 800m progressive effort with technical focus
- **Pre-Set**: 1,000m IM and build work
- **Threshold Development**: 2,000m descending ladder at threshold
- **Race Pace 300s**: 1,800m at goal Olympic pace
- **Negative Split 200s**: 1,600m with back-half speed emphasis
- **Kick Race Pace**: 800m legs at race-specific intensity
- **Pull Power**: 800m upper body endurance
- **Cooldown**: 400m easy swimming

### Collectible Metrics
- Average pace vs. world-record pace
- Lactate response at race pace (if available)
- Stroke efficiency maintenance across sets
- Heart rate recovery profiles
- Subjective fatigue and effort ratings
