# ZombieFit.org Jul 2013

Welcome to the July collection! This month's programming focuses on integrating explosive power and foundational bodyweight movements, challenging athletes to maintain high intensity across varied disciplines—from running sprints to complex calisthenics circuits. Get ready to test your limits in both parkour agility and raw strength.
