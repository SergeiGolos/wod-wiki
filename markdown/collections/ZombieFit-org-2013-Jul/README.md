# ZombieFit Jul 2013

July 2013 features 2 ZombieFit workouts built around for time efforts. Programming stays focused on for time challenges throughout the month. Athletes can expect for time and tabata work across the training block.

The movement emphasis this month falls on sit ups, pull ups, air squats. Supporting elements include push ups, jumping pull ups, precision, creating a balanced mix of pushing, pulling, squatting, and parkour-specific patterns. All workouts are tiered across three levels—Étudiant (Beginner), Avancée (Advanced), and Traceur (Expert)—so scaling is built in from the ground up.

Standout themes this month include parkour-specific skill work, timed interval protocols. With only 2 workouts on record, this is a lighter month—useful for deload or supplementary training.
