## Warm Up

Run 1/4 mile | bike 1 mile | row 250m.

3 rounds of 10:
- 50m quadrupedal movement
- 5 foot precisions
- Push Ups
- Double Unders

## WOD

This WOD is NOT for time. Complete each set of exercises as quickly as possible, but rest as needed between sets. Complete each numbered set fully before moving to the next.

### Étudiant — Beginner

```wod
(21-15-9)
  Jumping Pull Ups
  Push Ups
  Burpees
```

### Avancée — Advanced

```wod
(21-15-9)
  Pull Ups
  Push Ups
  Air Squats
  Burpees
```

### Traceur — Expert

A thruster is a front squat that goes into a full press. Pick a challenging weight that you can perform at least 5 times continuously. Warm up the movement; don’t just pick an arbitrary weight and start the WOD.

```wod
(21-15-9)
  Pull Up Burpees
  Handstand Push Ups
  Thrusters
```
