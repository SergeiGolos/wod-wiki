## WOD

### Étudiant

This workout is structured as a series of Tabata intervals (20 seconds work / 10 seconds rest implied).

```wod
Tabata Air Squats
1:00 Rest
Tabata Box Jumps (12 inches)
1:00 Rest
Tabata Precisions (5 feet)
1:00 Rest
Tabata Jumping Pull Ups
```

### Avancée

This workout is structured as a series of Tabata intervals.

```wod
Tabata Quadrupedal Movement*
1:00 Rest
Tabata Box Jumps (24 inches)
1:00 Rest
Tabata Precisions (6 feet)
1:00 Rest
Tabata Pull Ups
```

### Traceur

This workout is structured as a series of Tabata intervals.

```wod
Tabata Quadrupedal Movement*
1:00 Rest
Tabata Box Jumps (30 inches)
1:00 Rest
Tabata Precisions (7 feet)
1:00 Rest
Tabata L-sit Pull Ups
```

## Post

Post total number of repetitions performed to comments.

**For the tabata QM, use each twenty second period to go as far as possible while performing the quadrupedal movement. Score yourself by tabulating feet traveled while QMing.**
