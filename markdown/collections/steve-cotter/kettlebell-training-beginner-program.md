## Workout 1: Kettlebell Training - Beginner Program

**Category**: Steve Cotter Beginner  
**Type**: Foundation Building  
**Difficulty**: Beginner

### Description

This beginner program from Steve Cotter's "Kettlebell Training" book introduces fundamental movement patterns progressively. The program starts with the deadlift and swing before advancing to more complex movements like the clean and press. Each session builds on the previous one, ensuring solid foundations.

The program uses a structured approach with clear progression criteria. Movements are practiced at sub-maximal intensities to develop technique before adding load. The focus is on building the movement patterns that will support advanced training later.

### Week 1-2

```wod
(5)
  8 KB Deadlift 16kg
  :30 Rest

(5)
  5 Two-Hand KB Swing 16kg
  :30 Rest

(3)
  5 Goblet Squat 16kg
  :30 Rest
```
### Week 3-4

```wod
(5)
  10 Two-Hand KB Swing 16kg
  :30 Rest

(3)
  8 Goblet Squat 16kg
  :30 Rest

(3)
  5 Clean 16kg
  :45 Rest
```


### Breakdown

- **Rounds/Duration**: 4-week progressive program
- **Exercises**: Deadlift, Swing, Goblet Squat, Clean
- **Goal**: Establish fundamental movement patterns

### Progression Criteria

| Exercise | Beginner | Ready to Progress |
|----------|----------|-------------------|
| Deadlift | 16kg, 5 sets | 10 reps, perfect form |
| Swing | 16kg, 2-hand | 20 reps continuous |
| Goblet Squat | 16kg | Below parallel, neutral spine |

### Collectible Metrics

- Weight progression
- Rep quality
- Movement confidence
