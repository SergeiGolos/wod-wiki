## Warm Up

Run 1/4 mile | bike 1 mile | row 250m followed by:

```wod
(3)
  25m Quadrupedal Movement
  10 Sit Ups
  10 Push Ups
  10 Jumping Pull Ups
```

## WOD

### Étudiant — Beginner

```wod
(3)
  400m Run
  25 Lunges (alternating legs, so 25 total)
```

### Avancée — Advanced

```wod
(3)
  400m Run
  25 Lunges each leg
```

### Traceur — Expert

```wod
(3)
  400m Run
  25 Pistols | Single-Leg Squats (alternating legs, so 25 total)
```
