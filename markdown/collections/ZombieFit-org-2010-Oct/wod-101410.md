## Warm Up

Run 1/4 mile or bike 1 mile or row 250m followed by:

```wod
(3)
  25m Quadrupedal Movement
  10 Sit Ups
  10 Push Ups
```

## WOD

### Étudiant — Beginner

Perform 40 repetitions each of: dips, jumping jacks, air squats, sit ups, and jumping pull ups. Complete each exercise in as few sets as possible — go to failure on every set.

```wod
40 Dips
40 Jumping Jacks
40 Air Squats
40 Sit Ups
40 Jumping Pull Ups
```

### Avancée — Advanced

Perform 60 repetitions each of: dips, jumping jacks, air squats, sit ups, and jumping pull ups. Complete each exercise in as few sets as possible — go to failure on every set.

```wod
60 Dips
60 Jumping Jacks
60 Air Squats
60 Sit Ups
60 Jumping Pull Ups
```

### Traceur — Expert

Perform 75 repetitions each of: knees to elbows, push ups, air squats, and jumping pull ups. Complete each exercise in as few sets as possible — go to failure on every set. (Reference: knees-to-elbows demo video from CrossFit HQ.)

```wod
75 Knees to Elbows
75 Push Ups
75 Air Squats
75 Jumping Pull Ups
```
