# Syntax: AMRAP (Time Cap)

AMRAP stands for **As Many Rounds As Possible**. You define a fixed time duration, and the timer counts **down**.

Use a time (e.g., `20:00`) followed by `(AMRAP)`.

```wod
20:00
  (AMRAP)
    5 Pullups
    10 Pushups
    15 Squats
```

You can also use a simple time cap for a "For Time" workout to stop it if you take too long:

```wod
15:00 Time Cap
  Row 2000m
  50 Wall Balls
```
