# Syntax: Supplemental Data

Beyond movements and metrics, you can capture intent, effort, and auxiliary actions.

## Trends
Use `^` to indicate increasing weight or intensity over sets.
```wod
5 Deadlift ?lb
```

## Effort / RPE
You can log effort levels or descriptive notes using `@`.
```wod
Run 5km
Row 500m
```

## Actions and Rest
Special non-movement actions are enclosed in brackets `[]`.
```wod
5 Rounds
  Run 400m
  2:00 Rest
  Setup Barbell
```
