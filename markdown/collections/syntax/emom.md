# Syntax: EMOM (Every Minute on the Minute)

In an EMOM, the timer counts down for each minute and resets. You must complete the work before the minute is up.

Specify the total duration followed by `(EMOM)`.

```wod
10:00
  (EMOM)
    3 Clean & Jerk 135lb
```

You can also use larger intervals:

```wod
20:00
  Every 2:00
    Run 400m
```
