# Syntax: Tabata / Intervals

Intervals use a combination of work and rest timers.

A standard Tabata is 8 rounds of 20s work and 10s rest.

```wod
(8 Rounds)
  :20 Work
  :10 Rest
  Air Squats
```

You can define custom intervals:

```wod
(5 Rounds)
  3:00 Run 800m
  2:00 Rest
```
