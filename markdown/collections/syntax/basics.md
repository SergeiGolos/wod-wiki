# Syntax: The Basics

A WOD block is defined by ` ```wod ... ``` `. Inside, you list exercises and instructions. Each line represents a movement or a command.

```wod
Pushups
Situps
Squats
```

## Structure
- **Fences**: Use triple backticks followed by `wod`, `log`, or `plan`.
- **Exercises**: One per line.
- **Indentation**: Used for grouping and nested loops (see Groups section).
