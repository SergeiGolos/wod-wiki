# Syntax: Groups

Use parentheses `()` to group exercises into rounds or named sections. Indentation is required for exercises within a group.

## Simple Rounds
```wod
(3 Rounds)
  10 Pushups
  10 Situps
```

## Named Groups
You can name your groups for clarity (e.g., Warmup, Strength, Cool-down).
```wod
(Warmup)
  Run 400m
  Stretch
```

## Nested Groups
Groups can be nested to create complex structures.
```wod
(3 Rounds)
  (Tabata)
    :20 Work
    :10 Rest
    Air Squats
  1:00 Rest
```
