# Syntax: Measurements

Specify weights, distances, and other metrics directly in your exercise lines.

## Weights and Resistance
Specify weights using `lb`, `kg`, or `bw` (bodyweight).
```wod
Back Squat 225lb
Deadlift 100kg
Weighted Pullup 20lb
Air Squat bw
```

## Percentages
You can indicate percentages of 1RM or max effort.
```wod
Bench Press @75%
```

## Distance and Cardio
Supported units: `m`, `km`, `ft`, `miles`.
```wod
Run 400m
Row 2000m
Bike 10 miles
Swim 500m
```
