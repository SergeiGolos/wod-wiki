## Warm Up

Run 1/4 mile or bike 1/2 mile or row 500m followed by:

```wod
(2)
  75m Quadrupedal Movement
  50 Jumping Jacks
```

## WOD

### Étudiant — Beginner

Max effort for 30 seconds for each exercise. No rest between exercises. After each full round take 2:00. Perform 3 rounds total. Count total reps for all exercises and rounds.

```wod
(3)
  :30
    Push Ups
    Sit Ups
    Air Squats
    Jumping Pull Ups
  2:00 Rest
```

### Avancée — Advanced

Max effort for 30 seconds for each exercise. No rest between exercises. After each full round take 2:00. Perform 5 rounds total. Count total reps for all exercises and rounds.

```wod
(5)
  :30
    Push Ups
    Sit Ups
    Air Squats
    Pull Ups
  2:00 Rest
```
