## Warm Up

Run 1/4 mile | bike 1/2 mile | row 250m, followed by:

```wod
(3)
  10 Double Unders
  50m Quadrupedal Movement
```

## WOD

### Étudiant — Beginner

Perform each movement for 40 repetitions. Complete each exercise in as few sets as possible (go to failure on every set).

```wod
40 Dips
40 Jumping Jacks
40 Air Squats
40 Sit Ups
40 Jumping Pull Ups
```

### Avancée — Advanced

Perform each movement for 60 repetitions. Complete each exercise in as few sets as possible (go to failure on every set).

```wod
60 Dips
60 Jumping Jacks
60 Air Squats
60 Sit Ups
60 Pull Ups
```

### Traceur — Expert

Perform each movement for 75 repetitions. Complete each exercise in as few sets as possible (go to failure on every set).

```wod
75 Dips
75 Jumping Jacks
75 Elbows to Knees
75 Air Squats
75 Pull Ups
```
