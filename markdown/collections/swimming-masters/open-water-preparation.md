# Open Water Preparation

**Type**: Open Water Training  
**Duration**: 60-75 minutes  
**Difficulty**: Intermediate  

### Description
This workout prepares masters swimmers for open water events ranging from 1K to 5K distances. The session develops sighting skills, pack swimming awareness, and the sustained pacing required for open water racing. Pool-based exercises simulate race conditions while building the confidence needed for open water competition. Swimmers should practice bilateral breathing and varied tempo throughout.

```wod
(3) Warmup
  150m Freestyle
  :30 Rest

(6) Sighting Practice
  + 50m Sighting Swim
  + :20 Rest

(4) Drafting Set
  100m Draft Practice
  :30 Rest

(8) Steady State
  100m Freestyle
  :15 Rest

(4) Variable Tempo
  + 25m Fast
  + :10 Rest
  + 50m Moderate
  + :10 Rest
  + 25m Fast
  + :30 Rest

(4) Finish Simulation
  75m Build
  :30 Rest

150m Cooldown
```

### Breakdown
- **Warm-up**: 450m easy swimming
- **Sighting Practice**: 300m lifting head to sight without breaking rhythm
- **Drafting Set**: 400m practicing positioning behind other swimmers
- **Steady State**: 800m sustained aerobic effort with minimal rest
- **Variable Tempo**: 400m simulating race surges and recovery
- **Finish Simulation**: 300m building to strong finish
- **Cooldown**: 150m easy swimming

### Collectible Metrics
- Sighting accuracy and efficiency
- Pace consistency on steady state set
- Heart rate recovery patterns
