# Stroke Refinement Workshop

**Type**: Technique Focus  
**Duration**: 60 minutes  
**Difficulty**: All Levels  

### Description
This technique-focused session emphasizes stroke mechanics and efficiency over fitness. Each segment isolates specific aspects of all four competitive strokes using progressive drill sequences. This workout is ideal for masters swimmers returning to the sport after a break, those recovering from injury, or anyone seeking to improve their swimming economy. Coaches should provide individual feedback throughout the session.

```wod
(3) Warmup
  100m Freestyle
  :30 Rest

(4) Freestyle Mechanics
  + 25m Catch-Up Drill
  + :15 Rest
  + 25m Fingertip Drag
  + :15 Rest
  + 50m Freestyle Swim
  + :30 Rest

(4) Backstroke Technique
  + 25m Single Arm
  + :15 Rest
  + 25m Rotation Drill
  + :15 Rest
  + 50m Backstroke Swim
  + :30 Rest

(4) Breaststroke Timing
  + 25m Kick Only
  + :15 Rest
  + 25m 2-Kick 1-Pull
  + :15 Rest
  + 50m Breaststroke Swim
  + :30 Rest

(4) Butterfly Development
  + 25m Dolphin Kick
  + :15 Rest
  + 25m Single Arm Fly
  + :15 Rest
  + 50m Butterfly Swim
  + :30 Rest

100m Cooldown
```

### Breakdown
- **Warm-up**: 300m easy freestyle
- **Freestyle Mechanics**: 400m catch position and recovery work
- **Backstroke Technique**: 400m rotation and arm path development
- **Breaststroke Timing**: 400m kick-pull coordination
- **Butterfly Development**: 400m rhythm and timing
- **Cooldown**: 100m easy swimming

### Collectible Metrics
- Stroke count per length for each stroke
- Perceived comfort rating (1-5) per stroke
- Specific technique improvements identified
