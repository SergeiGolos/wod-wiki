## Workout 5: Fundamentals Program

**Category**: Joe Daniels Beginner  
**Type**: Skill Development  
**Difficulty**: Beginner

### Description

The Fundamentals Program is Joe Daniels' entry-level program designed to teach proper kettlebell technique from the ground up. The program covers the essential movements: swing, goblet squat, clean, press, and Turkish get-up. Each session builds on the previous one, ensuring solid foundations before progressing.

The program is designed for those new to kettlebells or those wanting to refine their technique. Sessions are shorter and focused on quality over quantity, establishing the movement patterns that will support long-term progress.

```wod
Fundamentals Session
(3)
  5 Two-Hand KB Swing 16kg
  :30 Rest

(3)
  5 Goblet Squat 16kg
  :30 Rest

(3)
  5 KB Deadlift 16kg
  :30 Rest

(2)
  3 Turkish Get Up Left 12kg
  3 Turkish Get Up Right 12kg
  :60 Rest
```

### Breakdown

- **Rounds/Duration**: 20-25 minutes
- **Exercises**: Swing, Goblet Squat, Deadlift, Turkish Get-Up
- **Goal**: Establish proper technique

### Collectible Metrics

- Movement quality
- Weight progression
- Confidence with movements
