## Workout 6: Sport Preparation Session

**Category**: Joe Daniels Sport Training  
**Type**: Sport-Specific Endurance  
**Difficulty**: Advanced

### Description

This session prepares athletes for kettlebell sport competition, focusing on the ability to maintain lifts for extended durations. The workout uses sport-style technique with relaxation between repetitions to build the efficiency necessary for 10-minute sets.

The session includes both jerk and snatch practice, following the structure of kettlebell sport competition. Rest periods are strategic, allowing for quality work while building the specific endurance required for competition.

### Sport Preparation

```wod
10:00 ? KB Jerk 16kg
5:00 Rest
10:00 ? KB Snatch 16kg
```
### Technical Practice

```wod
(5)
  1:00 KB Clean 16kg
  :30 Rest
```


### Breakdown

- **Rounds/Duration**: 25-30 minutes
- **Exercises**: Jerk, Snatch, Clean
- **Goal**: Build sport-specific endurance

### Collectible Metrics

- Total reps per 10-minute set
- Pace consistency
- Technique efficiency
