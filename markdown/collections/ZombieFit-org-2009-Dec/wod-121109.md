## Warm Up

Warm up options include:
*   Run 1/4 mile
*   Bike 2 miles
*   Row 500m

Followed by:
```wod
(3)
  10 Air Squats
  10 Sit Ups
  10 Jumping Pullups (May sub with Pushups if no pull up bar)
```

## WOD

### Étudiant

```wod
(5)
  50m Quadrupedal Movement
  5 x 5ft Precision Jumps
  10 Dips
```

### Avancée

```wod
(5)
  50m Quadrupedal Movement
  5 x 7ft Precision Jumps
  20 Dips
```

### Traceur

```wod
(5)
  100m Quadrupedal Movement
  5 x 8ft Precision Jumps
  30 Dips
```

## Cool Down

Complete 25 pull ups (May be dead hang, kipping, jumping or assisted).
