## Warm Up

Jog 1/2 mile OR row 1000m OR bike 5 miles.

## WOD

### Étudiant — Beginner

Perform the following movements for time:

```wod
(3)
  (6-4-2-4-6)
    Push Ups
    Sit Ups
    Squats
    Jumping Pull Ups
```

### Avancée — Advanced

Perform the following movements for time:

```wod
(3)
  (10-8-6-4-2-4-6-8-10)
    Push Ups
    Sit Ups
    Double Unders
    Pull Ups
```

### Traceur — Expert

Perform the following movements for time:

```wod
(3)
  (10-8-6-4-2-4-6-8-10)
    Handstand Push Ups
    Sit Ups
    Double Unders
    Pull Ups
```

## Cool Down

Sideways quadrupedal movement 50m both ways.
