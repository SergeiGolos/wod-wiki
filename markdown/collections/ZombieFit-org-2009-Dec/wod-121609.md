## WOD

### Étudiant

```wod
(3)
  6-4-2-4-6 Push ups / Sit ups / Squats / Jumping Pullups
```

### Avancée

```wod
(3)
  10-8-6-4-2-4-6-8-10 Push ups / Sit ups / Double Unders / Pullups
```

### Traceur

```wod
(3)
  10-8-6-4-2-4-6-8-10 Handstand Pushups / Sit ups / Double Unders / Pullups
```
