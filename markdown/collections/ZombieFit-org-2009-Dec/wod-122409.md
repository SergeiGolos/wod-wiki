## WOD

### Étudiant

This workout is a progressive AMRAP structure, increasing reps each minute until failure.

**Progression:**
```wod
(N) :? EMOM / 1 Air Squat / 1 Sit Up / 1 Push Up
+ (2) :? EMOM / 2 Air Squats / 2 Sit Ups / 2 Push Ups
... continue increasing reps each minute until failure.
```

### Avancée

This workout is a progressive AMRAP structure, increasing reps each minute until failure.

**Progression:**
```wod
(N) :? EMOM / 1 Air Squat / 1 Sit Up / 1 Push Up / 1 Pull Up
+ (2) :? EMOM / 2 Air Squats / 2 Sit Ups / 2 Push Ups / 2 Pull Ups
```

### Traceur

This workout is a progressive AMRAP structure, increasing reps each minute until failure.

**Progression:**
```wod
(N) :? EMOM / 1 Air Squat / 1 Sit Up / 1 Push Up / 1 Double Unders / 1 Pull Up
+ (2) :? EMOM / 2 Air Squats / 2 Sit Ups / 2 Push Ups / 2 Double Unders / 2 Pull Ups
... continue increasing reps each minute until failure.
```

## Cool Down

This section is completed at own pace and is not a metabolic conditioning workout.

*   **Completed < 5 rounds:** Perform 50 repetitions of each movement.
*   **Completed 5 - 10 rounds:** Perform 25 repetitions of each movement.
*   **Completed 10 - 15 rounds:** Perform 10 repetitions of each movement.
*   **Completed ≥ 15 rounds:** No additional work required.
