## Warm Up

Run 1/4 mile or bike 2 miles or row 250m.

```wod
(3)
  10 Air Squats
  10 Sit Ups
  10 Push Ups
  10 Pull Ups
```

## WOD

### Étudiant — Beginner

Increase the number of repetitions each minute by 1. On minute 1 perform 1 rep of each movement, on minute 2 perform 2 reps of each, etc. Continue until you fail to complete the required repetitions within the minute.

```wod
:60 EMOM
  + 1 Air Squat // add 1 rep each minute
  + 1 Push Up // add 1 rep each minute
  + 1 Sit Up // add 1 rep each minute
```

### Traceur — Expert

Increase the number of repetitions each minute by 1. On minute 1 perform 1 rep of each movement, on minute 2 perform 2 reps of each, etc. Continue until you fail to complete the required repetitions within the minute.

```wod
:60 EMOM

  + 1 Sit Up // add 1 rep each minute
  + 1 Push Up // add 1 rep each minute
  + 1 Double Under // add 1 rep each minute
  + 1 Pull Up // add 1 rep each minute
```

## Cool Down

If you completed fewer than 5 rounds, complete 50 repetitions of each movement. If you completed 5–9 rounds, complete 25 repetitions of each movement. If you completed 10–14 rounds, complete 10 repetitions of each movement. If you completed 15 rounds or more, no additional work is required. This is to be completed on your own time; it is not a metabolic conditioning workout.
