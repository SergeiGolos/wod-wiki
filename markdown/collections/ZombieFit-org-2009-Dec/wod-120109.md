## Warm Up

Run 1/4 mile OR bike 2 miles OR row 500m followed by:

```wod
(2)
  250m Quadrupedal Movement
  50 Jumping Jacks
```

## WOD

### Étudiant — Beginner

Max effort for 30 seconds for each exercise. Count total reps for all exercises and rounds. Rest 2 minutes between rounds.

```wod
(3)
  :30 Push Ups
  :30 Sit Ups
  :30 Air Squats
  :30 Jumping Pull Ups
  2:00 Rest
```

### Avancée — Advanced
