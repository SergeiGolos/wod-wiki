## Warm Up

Run 1/4 mile | bike 1/2 mile | row 500m followed by:

```wod
(3)
  25m Quadrupedal Movement
  10 Push Ups
  10 Sit Ups
  10 Air Squats
  25 Jumping Jacks
```

## WOD

### Étudiant — Beginner

```wod
(5)
  100m Backwards Run
  15 Push Ups
  15 Air Squats
```

### Avancée — Advanced

```wod
(5)
  100m Backwards Run
  25 Push Ups
  25 Air Squats
```

### Traceur — Expert

```wod
(5)
  200m Backwards Run
  50 Push Ups
  50 Air Squats
```

## Cool Down

In as few sets as possible, perform 25 Five-Foot Precisions.

```wod
25 Five-Foot Precisions
```
