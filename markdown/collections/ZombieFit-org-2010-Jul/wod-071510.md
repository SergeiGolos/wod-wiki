## Warm Up

Run 1/4 mile | bike 2 miles | row 500m followed by:

```wod
(3)
  10 Pull Ups
  10 Push Ups
  10 Parkour Squats
  10 Sit Ups
```

## WOD

### Étudiant — Beginner

```wod
50 Burpees
```

### Avancée — Advanced

```wod
100 Burpees
```

### Traceur — Expert

If you do not have a pull up bar available, perform precision burpees: add a short broad jump (about 5–6 feet) to the end of each burpee instead of a pull-up.

```wod
100 Burpee Pull Ups
```

## Cool Down

```wod
(3)
  50m Reverse Quadrupedal Movement
```
