## Warm Up

Run 1/4 mile | row 250m | bike 1/2 miles.

```wod
(3)
  10 Jumping Jacks
  10 Sit Ups
  10 Pull Ups // may sub Jumping Pull Ups
  10 Air Squats
```

## WOD

### Étudiant — Beginner

```wod
(21-15-9)
  12 inch Box Jumps
  Push Ups
  Sit Ups
```

### Avancée — Advanced

```wod
(21-15-9)
  24 inch Box Jumps
  Push Ups
  Knees to Elbows
```

### Traceur — Expert

```wod
(21-15-9)
  36 inch Box Jumps
  Push Ups
  Knees to Elbows
```

If needed, sub parkour squats for box jumps.
