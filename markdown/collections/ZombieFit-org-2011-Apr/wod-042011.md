## Warm Up

Run 1/4 mile OR row 250m OR bike 1/2 miles.

On the minute, for 5 minutes, sprint 100m.

## WOD

### Étudiant — Beginner

Tabata air squats, 1 minute rest, tabata box jumps (12in), 1 minute rest, tabata 5-foot precisions, 1 minute rest, tabata jumping pull ups.

```wod
(8) :20 Air Squats
1:00 Rest
(8) :20 Box Jumps 12in
1:00 Rest
(8) :20 5-Foot Precisions
1:00 Rest
(8) :20 Jumping Pull Ups
```

### Avancée — Advanced

Tabata quadrupedal movement, 1 minute rest, tabata box jumps (24in), 1 minute rest, tabata 6-foot precisions, 1 minute rest, tabata pull ups.

```wod
(8) :20 Quadrupedal Movement
1:00 Rest
(8) :20 Box Jumps 24in
1:00 Rest
(8) :20 6-Foot Precisions
1:00 Rest
(8) :20 Pull Ups
```

### Traceur — Expert

Tabata quadrupedal movement, 1 minute rest, tabata box jumps (30in), 1 minute rest, tabata 7-foot precisions, 1 minute rest, tabata L-sit pull ups.

```wod
(8) :20 Quadrupedal Movement
1:00 Rest
(8) :20 Box Jumps 30in
1:00 Rest
(8) :20 7-Foot Precisions
1:00 Rest
(8) :20 L-Sit Pull Ups
```

## Cool Down

Go home and watch some parkour videos.

## Post

Post total number of repetitions performed to comments.

* For the tabata Quadrupedal Movement, use each twenty second period to go as far as possible while performing the movement. Score yourself by tabulating feet traveled while QMing.
** As always, you may sub tuck jumps for box jumps.
