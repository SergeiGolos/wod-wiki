# ZombieFit Jul 2011

July 2011 features 5 ZombieFit workouts built around for time efforts. Programming rotates through for time, amrap, skill and intervals formats to keep sessions varied and progressive. Athletes can expect for time, amrap, max effort, skill work and intervals work across the training block.

The movement emphasis this month falls on push ups, pull ups, precision. Supporting elements include sit ups, quadrupedal movement, air squats, creating a balanced mix of pushing, pulling, squatting, and parkour-specific patterns. All workouts are tiered across three levels—Étudiant (Beginner), Avancée (Advanced), and Traceur (Expert)—so scaling is built in from the ground up.

Standout themes this month include parkour-specific skill work, density-based AMRAP sessions. The 5 sessions form a solid training block with steady volume and progressive demand.
