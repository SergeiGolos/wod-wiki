## Warm Up

Run 1/4 mile | row 250m | bike 1/2 mile.

3 rounds of:
- 25m Quadrupedal Movement
- 10 Squats
- 10 Sit Ups
- 10 Double Unders (or 50 Single Unders)
- 10 Push Ups

## WOD

### Étudiant — Beginner

```wod
(15-11-9)
  20 inch Box Jumps
  Jumping Pull Ups
  Sit Ups
  Push Ups
```

### Avancée — Advanced

```wod
(21-15-9)
  24 inch Box Jumps
  Pull Ups
  Sit Ups
  Push Ups
```

### Traceur — Expert

```wod
(21-15-9)
  30 inch Box Jumps
  Pull Ups
  Sit Ups
  Handstand Push Ups
```
