## Warm Up

Run 1/2 mile | bike 1/2 mile | row 250m.

```wod
(3)
  10 Parkour Squats | Air Squats
  20 Sit Ups
  75ft Quadrupedal Movement
```

## WOD

### Étudiant — Beginner

```wod
(3)
  Push Ups to failure
  200 Push Ups // knees   # scale: if you cannot do full push-ups
```

### Avancée — Advanced

```wod
(4)
  Push Ups to failure
```

### Traceur — Expert

```wod
(5)
  Push Ups to failure
Rest 5:00
(1)
  Pull Ups to failure
```

## Cool Down

5 minute jog and a stretch
