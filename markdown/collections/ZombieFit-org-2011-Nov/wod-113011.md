## Warm Up

Run 1/4 mile OR bike 0.5 miles OR row 250m followed by:

```wod
(3)
  10 Sit Ups
  10 Push Ups
  10 Air Squats
  10 Pull Ups / Handstand Push Ups
```

## WOD

### Étudiant — Beginner

As fast as possible, complete the following (may use any order and any number of sets, just complete all movements):

```wod
25 Push Ups
25 Sit Ups
200 Jumping Jacks
50 Jumping Pull Ups
25 Five-Foot Precisions
```

### Avancée — Advanced

As fast as possible, complete the following (may use any order and any number of sets, just complete all movements):

```wod
50 Push Ups
50 Sit Ups
300 Jumping Jacks
50 Five-Foot Precisions
```

### Traceur — Expert

As fast as possible, complete the following (may use any order and any number of sets, just complete all movements):

```wod
75 Push Ups
75 Sit Ups
300 Jumping Jacks
75 Pull Ups
75 Five-Foot Precisions
```
