November 2011 features 2 ZombieFit workouts built around for time efforts. Programming stays focused on for time challenges throughout the month. Athletes can expect consistent for time work across the training block.

The movement emphasis this month falls on push ups, precision, sit ups. Supporting elements include pull ups, run, dips, creating a balanced mix of pushing, pulling, squatting, and parkour-specific patterns. All workouts are tiered across three levels—Étudiant (Beginner), Avancée (Advanced), and Traceur (Expert)—so scaling is built in from the ground up.

Standout themes this month include parkour-specific skill work. With only 2 workouts on record, this is a lighter month—useful for deload or supplementary training.
