## Warm Up

Run 1/4 mile | bike 1/2 mile | row 250m followed by:

```wod
(3)
  10 Sit Ups
  10 Push Ups
  10 Air Squats
  10 Pull Ups | Handstand Push Ups
```

## WOD

In as few sets as possible, complete the following:

### Étudiant — Beginner

```wod
40 Dips
40 Six-Foot Precisions
50 Alternating Leg Lunges
1/4 Mile Run
```

### Avancée — Advanced

```wod
60 Dips
60 Six-Foot Precisions
60 Alternating Leg Lunges
1/2 Mile Run
```

### Traceur — Expert

```wod
75 Dips
75 Six-Foot Precisions
75 Alternating Leg Lunges
1/2 Mile Run
```

## Post

Post # of sets to comments.

This WOD is meant to be completed in as few sets as possible, which means you can rest as long as you like in between sets. Suggested rest: try not to rest more than 2 minutes. You do NOT have to complete these movements in any order — e.g., do 20 dips, then 20 precisions, etc. Try to go to failure on every set.
