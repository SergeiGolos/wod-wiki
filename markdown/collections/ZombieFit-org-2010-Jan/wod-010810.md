## Warm Up

Run 1/4 mile | bike 2 miles | row 250m.

```wod
(3)
  10 Double Unders
  10 Jumping Push Ups
  10 Sit Ups
  10 Dips
```

## WOD

### Étudiant — Beginner

Perform the following movements for time:

```wod
(21-15-9)
  Burpees
  Push Ups
  Sit Ups
```

### Avancée — Advanced

Perform the following movements for time:

```wod
(21-15-9)
  Burpees
  Push Ups
  Sit Ups
```

### Traceur — Expert

Perform the following movements for time. In between each round perform 50m quadrupedal movement.

```wod
(21-15-9)
  Burpees
  Push Ups
  Sit Ups
  50m Quadrupedal Movement
```

## Cool Down

```wod
100 Deep Knee Air Squats
```
