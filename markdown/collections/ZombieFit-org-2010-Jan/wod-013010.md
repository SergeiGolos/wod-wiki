## Warm Up

Warm up options include:
*   Run 1/4 mile
*   Bike 2 miles
*   Row 250m

Followed by a circuit of 3 rounds:
*   50m quadrupedal movement
*   10 jumping jacks
*   5 burpees

## WOD

### Étudiant
Complete the following Tabata protocol for 4 sets (instead of 8) per exercise, with no rest between movements. Keep track of total reps completed across all exercises.

```wod
(4) :20 ? Air Squats / :10 Rest
(4) :20 ? Push Ups / :10 Rest
(4) :20 ? Sit Ups / :10 Rest
(4) :20 ? Pull Ups / :10 Rest
```

### Avancée
Complete the full Tabata protocol for 8 sets per exercise, with no rest between movements. Substitutions are allowed (e.g., jumping pull ups or push ups on knees). Keep track of total reps completed across all exercises.

```wod
(8) :20 ? Air Squats / :10 Rest
(8) :20 ? Push Ups / :10 Rest
(8) :20 ? Sit Ups / :10 Rest
(8) :20 ? Pull Ups / :10 Rest
```

### Traceur
Complete the full Tabata protocol for 8 sets per exercise, maintaining correct form and full range of motion for every movement. Keep track of total reps completed across all exercises.

```wod
(8) :20 ? Air Squats / :10 Rest
(8) :20 ? Push Ups / :10 Rest
(8) :20 ? Sit Ups / :10 Rest
(8) :20 ? Pull Ups / :10 Rest
```

## Cool Down

3 sets of 5 parkour squats.
