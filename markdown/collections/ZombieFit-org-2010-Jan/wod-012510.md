## Warm Up

Run 1/4 mile | bike 2 miles | row 250m followed by:

```wod
(3)
  10 Sit Ups
  10 Push Ups
  10 Air Squats
  10 Pull Ups
```

May substitute jumping pull-ups for pull-ups.

## WOD

### Étudiant — Beginner

Perform the following for time:

```wod
(10-8-6-4-2-4-6-8-10)
  Burpees
  Sit Ups
```

### Avancée — Advanced

Perform the following for time:

```wod
(10-8-6-4-2-4-6-8-10)
  Burpees
  Sit Ups
  Double Unders
```

### Traceur — Expert

Perform the following for time:

```wod
(10-8-6-4-2-4-6-8-10)
  Burpee Pull Ups
  Sit Ups
  Double Unders
```

## Cool Down

In the lowest number of sets, perform 50 push ups.

```wod
50 Push Ups
```
