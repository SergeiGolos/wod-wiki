## WOD

### Étudiant

For Time:
```wod
(2-4-6-8-10)
  Burpees
  Five Foot Precisions
```

### Avancée

Same as Étudiant, plus an extra set of 12 (e.g., 12-2-10, etc.) and using six foot precisions:
```wod
(2-4-6-8-10)
  Burpees
  Six Foot Precisions
```

### Traceur

Same as Avancée, but replacing burpees with pull-up burpees:
```wod
(2-4-6-8-10)
  Pull-Up Burpees
  Six Foot Precisions
```
