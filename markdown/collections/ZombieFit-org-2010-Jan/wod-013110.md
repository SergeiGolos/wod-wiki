## Warm Up

Run 1/4 mile or bike 2 miles or row 250m followed by:

```wod
(3)
  10 Push Ups
  10 Sit Ups
  10 Air Squats
  10 Pull Ups
```

## WOD

### Étudiant — Beginner

Perform the following for time:

```wod
(21-15-9)
  Single Unders
  5-Foot Precisions // focus on technique
  Lunges // alternate legs
```

### Avancée — Advanced

Perform the following for time:

```wod
(21-15-9)
  Double Unders
  6-Foot Precisions
  Lunges (each leg)
```

### Traceur — Expert

Perform the following for time:

```wod
(21-15-9)
  Double Unders
  6-Foot Precisions
  Pistols // alternate legs
```

## Cool Down

In as few sets as possible, perform 150 air squats.

```wod
150 Air Squats
```
