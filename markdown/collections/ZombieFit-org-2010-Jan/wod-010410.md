## WOD

### Étudiant
This session focuses on practicing the press progression movements using a broomstick or PVC pipe. Pause and correct form after every repetition.

*   **Shoulder Press:** 50 repetitions
*   **Push Press:** 50 repetitions
*   **Push Jerk:** 50 repetitions

### Avancée
Work on form with light weight. Complete the following sets for each movement:

*   **Shoulder Press Sets:** 5, 5, 3, 3, 1, 1, 1
*   **Push Press Sets:** 5, 5, 3, 3, 1, 1, 1
*   **Push Jerk Sets:** 5, 5, 3, 3, 1, 1, 1

### Traceur
Work towards your one rep max. Complete the following sets for each movement:

*   **Shoulder Press Sets:** 5, 5, 3, 3, 1, 1, 1
*   **Push Press Sets:** 5, 5, 3, 3, 1, 1, 1
*   **Push Jerk Sets:** 5, 5, 3, 3, 1, 1, 1
