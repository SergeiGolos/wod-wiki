## Warm Up

```wod
Run 1/4 mile | Bike 2 miles | Row 500m

(3)
  50m Quadrupedal Movement
  10 Air Squats
  10 Sit Ups
  10 Pull Ups
  10 Push Ups
```

## WOD

### Étudiant — Beginner

Practice the press movement with a broomstick | PVC; focus on technique. This is NOT a conditioning WOD.

```wod
50 Shoulder Press // broomstick or PVC
50 Push Press // broomstick or PVC
50 Push Jerk // broomstick or PVC
```

### Avancée — Advanced

If unfamiliar with any movement, perform at the Étudiant level. Keep weight very light if you require a spot; prioritize form.

```wod
(5-5-3-3-1-1-1)
  Shoulder Press

(5-5-3-3-1-1-1)
  Push Press

(5-5-3-3-1-1-1)
  Push Jerk
```

### Traceur — Expert

Work towards a one-rep max on each movement. Perform the listed set scheme.

```wod
(5-5-3-3-1-1-1)
  Shoulder Press

(5-5-3-3-1-1-1)
  Push Press

(5-5-3-3-1-1-1)
  Push Jerk
```

## Cool Down

```wod
(21-15-9)

  Air Squats
```
