# Sprint Distance Preparation

**Type**: Race-Specific  
**Duration**: 45-55 minutes  
**Difficulty**: Beginner-Intermediate  

### Description
This workout prepares triathletes for sprint-distance swim legs (typically 750m). The session emphasizes fast starts, sighting efficiency, and negative split pacing to ensure athletes exit the water with energy remaining for the bike and run. The main set simulates race effort with interval work that builds confidence at race pace. This session works well in the final 4-6 weeks before a sprint-distance race.

```wod
(3) Warmup
  100m Freestyle
  :20 Rest

(4) Sighting Drills
  25m Sighting Practice
  :15 Rest

(6) Race Pace Set
  100m Freestyle
  :30 Rest

(4) Negative Split
  150m Freestyle
  :30 Rest

(4) Fast Start Practice
  50m Sprint
  1:00 Rest

100m Cooldown
```

### Breakdown
- **Warm-up**: 300m easy swimming with progressive build
- **Sighting Drills**: 100m practicing head-up sighting every 6 strokes
- **Race Pace Set**: 600m at goal sprint race pace
- **Negative Split**: 600m with second half faster than first
- **Fast Start Practice**: 200m simulating mass start urgency
- **Cooldown**: 100m easy recovery

### Collectible Metrics
- Race pace times vs. goal race pace
- Sighting efficiency (strokes between sights)
- Perceived effort on fast starts
