# Ironman Distance Mastery

**Type**: Ultra-Endurance  
**Duration**: 90-120 minutes  
**Difficulty**: Advanced  

### Description
This comprehensive workout prepares triathletes for the Ironman swim (3.8km). The session combines long continuous swimming with race-pace intervals to build both endurance and confidence at Ironman distance. Special attention is given to energy management, sighting efficiency, and mental strategies for the 2.4-mile swim. This session should be completed during peak training phases 4-8 weeks before race day.

```wod
(3) Warmup
  200m Freestyle
  :30 Rest

(4) Technique Activation
  100m Drill Focus
  :20 Rest

(4) Sighting Mastery
  150m Sighting Swim
  :30 Rest

(6) Long Intervals
  300m Freestyle
  :45 Rest

(4) Race Pace Execution
  400m Freestyle
  1:00 Rest

(2) Race Simulation
  750m Continuous
  :30 Rest

(4) Negative Split Building
  250m Freestyle
  :30 Rest

300m Cooldown
```

### Breakdown
- **Warm-up**: 600m progressive effort
- **Technique Activation**: 400m efficiency focus
- **Sighting Mastery**: 600m establishing reliable sight pattern
- **Long Intervals**: 1,800m building distance capacity
- **Race Pace Execution**: 1,600m at Ironman goal pace
- **Race Simulation**: 1,500m continuous at race effort
- **Negative Split Building**: 1,000m practicing back-half strength
- **Cooldown**: 300m easy recovery

### Collectible Metrics
- Total distance completed
- Average pace across all main sets
- Stroke efficiency (distance per stroke)
- Perceived effort at race pace
