# Half Ironman (70.3) Building

**Type**: Long Distance  
**Duration**: 75-90 minutes  
**Difficulty**: Intermediate-Advanced  

### Description
This workout develops the endurance and pacing discipline required for 70.3 distance swims (1.9km). The session emphasizes energy conservation through efficient stroke mechanics and smart pacing. Extended continuous swimming builds the muscular endurance needed for the 1.2-mile swim while developing the mental fortitude for longer events. Athletes should practice steady, controlled effort that leaves them fresh for T1.

```wod
(3) Warmup
  200m Freestyle
  :30 Rest

(4) Sighting Rhythm
  100m Sighting Swim
  :20 Rest

(6) Building Endurance
  200m Freestyle
  :30 Rest

(8) Sustained Pace
  150m Freestyle
  :20 Rest

(4) Open Water Simulation
  300m Continuous
  :30 Rest

(4) Negative Split
  200m Freestyle
  :30 Rest

250m Cooldown
```

### Breakdown
- **Warm-up**: 600m easy progressive swimming
- **Sighting Rhythm**: 400m establishing sight pattern
- **Building Endurance**: 1,200m controlled effort
- **Sustained Pace**: 1,200m at goal race pace
- **Open Water Simulation**: 1,200m continuous with no lane lines
- **Negative Split**: 800m practicing back-half speed
- **Cooldown**: 250m easy swimming

### Collectible Metrics
- Average pace across sustained set
- Stroke count consistency at distance
- Perceived energy level after main set
