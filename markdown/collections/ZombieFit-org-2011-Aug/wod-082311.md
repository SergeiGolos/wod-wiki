## Warm Up

Run 1/4 mile OR bike 1 mile OR row 250m.

```wod
(3)
  25m Quadrupedal Movement
  10 Sit Ups
  10 Push Ups
  10 Jumping Pull Ups
```

## WOD

### Étudiant — Beginner

Complete each movement as Tabata (8 rounds: 20s on, 10s off). Rest 1:00 between Tabatas.

```wod
(8) :20 Tabata
  Air Squats
1:00 Rest
(8) :20 Tabata
  Box Jumps 12 inches
1:00 Rest
(8) :20 Tabata
  5-Foot Precisions
1:00 Rest
(8) :20 Tabata
  Jumping Pull Ups
```

## Post

Post total number of repetitions performed to comments.

For the Tabata QM, use each 20-second work period to go as far as possible while performing the quadrupedal movement. Score yourself by tabulating feet traveled while QMing.
