## Warm Up

Run 1/4 mile | bike 1 mile | row 250m

```wod
(3)
  50m Quadrupedal Movement
  5 Five-Foot Precisions
  10 Push Ups
  10 Double Unders
```

## WOD

This WOD is NOT for time. Complete each set of exercises as quickly as possible, but rest as much as needed in between each set. For instance, complete the first set of pull ups in the Étudiant workout as quickly as possible. Rest as needed before continuing onto the set of 21 burpees. When you start the next set complete all reps as fast as you can. Do this for all levels.

### Étudiant — Beginner

```wod
(21-15-9)
  Jumping Pull Ups
  Push Ups
  Burpees
```

### Avancée — Advanced

```wod
(21-15-9)
  Pull Ups
  Push Ups
  Air Squats
  Burpees
```

### Traceur — Expert

```wod
(21-15-9)
  Pull Up Burpees
  Handstand Push Ups
  Thrusters
```

A thruster is a front squat that goes into a full press. Pick a challenging weight that you can perform at least 5 times continuously. Warm up the movement; don’t just pick an arbitrary weight and start the WOD.
