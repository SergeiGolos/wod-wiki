## Warm Up

Warm up: jog 1/2 mile OR row 1000m OR bike 5 miles.

## WOD

### Étudiant — Beginner

3 rounds for time:

```wod
(3)
  (6-4-2-4-6)
    Push Ups
    Sit Ups
    Squats
    Jumping Pull Ups
```

### Avancée — Advanced

3 rounds for time:

```wod
(3)
  (10-8-6-4-2-4-6-8-10)
    Push Ups
    Sit Ups
    Double Unders
    Pull Ups
```

### Traceur — Expert

3 rounds for time:

```wod
(3)
  (10-8-6-4-2-4-6-8-10)
    Handstand Push Ups
    Sit Ups
    Double Unders
    Pull Ups
```

## Cool Down

```wod
Sideways QM 50m both ways
```
