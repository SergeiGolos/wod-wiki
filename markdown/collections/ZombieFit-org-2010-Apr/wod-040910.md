## Warm Up

Warm up options:
*   Run 1/4 mile
*   Row 250m
*   Bike 2 miles

Circuit (3 rounds):
```wod
(3)
  10 Squats
  10 Sit Ups
  10 Double Unders (or 50 Single Unders)
  10 Pushups
```

## WOD

### Étudiant

This workout consists of three Tabata circuits separated by 1 minute rest.

**Circuit 1:**
```wod
Tabata Air Squats
:20 ? Air Squats / :10 Rest
```
*Rest 1 minute*

**Circuit 2:**
```wod
Tabata Box Jumps (12 inches)
:20 ? Box Jumps / :10 Rest
```
*Rest 1 minute*

**Circuit 3:**
```wod
Tabata 5 foot Precisions
:20 ? Precisions / :10 Rest
```
*Rest 1 minute*

**Circuit 4:**
```wod
Tabata Jumping Pullups
:20 ? Pullups / :10 Rest
```
