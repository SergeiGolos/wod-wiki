## Warm Up

Run 1/4 mile | bike 2 miles | row 500m followed by:

3 rounds of:
- 50m quadrupedal movement
- 10 Jumping Jacks
- 5 Burpees

## WOD

Tabata This — 4 exercises performed in Tabata format (8 sets of 20s work / 10s rest each). Do all sets of one exercise before moving to the next. Count all repetitions; total reps across all sets is your score.

### Étudiant — Beginner

Complete a half Tabata (4 sets per exercise instead of 8).

```wod
(4) :30 Tabata
  Air Squats
  Push Ups
  Sit Ups
  Pull Ups
```

### Avancée — Advanced

Complete the full Tabata (8 sets per exercise). Use substitutions as needed (e.g., jumping pull-ups, knee push-ups).

```wod
(8) :30 Tabata
  Air Squats
  Push Ups
  Sit Ups
  Pull Ups
```

### Traceur — Expert

Complete the full Tabata (8 sets per exercise) using strict form and full range of motion for every movement.

```wod
(8) :30 Tabata
  Air Squats
  Push Ups
  Sit Ups
  Pull Ups
```
