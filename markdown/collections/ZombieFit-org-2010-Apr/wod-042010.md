## WOD

### Étudiant

Complete a 1/2 Tabata This workout. (4 sets per exercise).

```wod
(4) :20 ? Air Squats / :10 Rest
(4) :20 ? Push Ups / :10 Rest
(4) :20 ? Sit Ups / :10 Rest
(4) :20 ? Pull Ups / :10 Rest
```

### Avancée

Complete the full Tabata This workout (8 sets per exercise). Substitutions are allowed.

```wod
(8) :20 ? Air Squats / :10 Rest
(8) :20 ? Push Ups / :10 Rest
(8) :20 ? Sit Ups / :10 Rest
(8) :20 ? Pull Ups / :10 Rest
```

### Traceur

Complete the full Tabata This workout (8 sets per exercise), maintaining correct form and full range of motion.

```wod
(8) :20 ? Air Squats / :10 Rest
(8) :20 ? Push Ups / :10 Rest
(8) :20 ? Sit Ups / :10 Rest
(8) :20 ? Pull Ups / :10 Rest
```
