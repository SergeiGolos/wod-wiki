# ZombieFit Sep 2012

September 2012 features 1 ZombieFit workouts built around tabata efforts. Programming stays focused on intervals challenges throughout the month. Athletes can expect tabata and intervals work across the training block.

The movement emphasis this month falls on pull ups, box jumps, precision. Supporting elements include quadrupedal movement, air squats, jumping pull ups, creating a balanced mix of pushing, pulling, squatting, and parkour-specific patterns. All workouts are tiered across three levels—Étudiant (Beginner), Avancée (Advanced), and Traceur (Expert)—so scaling is built in from the ground up.

Standout themes this month include parkour-specific skill work, timed interval protocols. With only 1 workouts on record, this is a lighter month—useful for deload or supplementary training.
