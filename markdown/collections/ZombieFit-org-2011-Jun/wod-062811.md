## Warm Up

Run 1/4 mile | bike 1 mile | row 250m.

Followed by 3 rounds of:
- 10 air squats
- 10 sit-ups
- 10 push-ups
- 10 pull-ups

## WOD

### Étudiant — Beginner

```wod
(3)
  400m Run
  15 Push Ups
```

### Avancée — Advanced

```wod
(3)
  400m Run
  25 Push Ups
```

### Traceur — Expert

```wod
(3)
  400m Run
  50 Push Ups
```
