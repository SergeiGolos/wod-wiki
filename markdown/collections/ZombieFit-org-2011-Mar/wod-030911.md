## WOD

WOD 349 is a cumulative effort to be completed in as few sets as possible. The movements can be performed in any order.

### Étudiant

```wod
40 Dips
40 Six-Foot Precisions
100 Alternating Leg Lunges
1/4 mile Run
```

### Avancée

```wod
60 Dips
60 Six-Foot Precisions
100 Alternating Leg Lunges
1/4 mile Run
```

### Traceur

```wod
60 Dips
60 Six-Foot Precisions
200 Alternating Leg Lunges
1/2 mile Run
```

## Post

**Notes:**
*   This WOD is meant to be completed in as few sets as possible, which means you can rest as long as you like in between sets. (Suggested maximum rest: 2 minutes).
*   You DO NOT have to complete these movements in any order. You could do 20 dips, then move to 20 precisions, etc. Just try to go to failure on every set.
