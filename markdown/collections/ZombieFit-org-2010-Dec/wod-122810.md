## Warm Up

Run 1/4 mile | bike 1/2 mile | row 250m.

```wod
(3)
  25m Quadrupedal Movement
  10 Jumping Pull Ups
  10 Five-Foot Precisions
```

## WOD

### Étudiant — Beginner
Complete only 5 tabata sets.

```wod
(5) :20 Tabata
  Rowing
  Biking
  Running
```

### Avancée — Advanced
Complete a full tabata workout (8 sets).

```wod
(8) :20 Tabata
  Rowing
  Biking
  Running
```

### Traceur — Expert
Complete a full tabata workout (8 sets).

```wod
(8) :20 Tabata
  Rowing
  Biking
  Running
```

## Cool Down

Walk 1/2 mile.
