## Warm Up

Run 1/4 mile OR row 250m OR bike 0.5 miles.

```wod
(3)
  25m Quadrupedal Movement
  10 Push Ups
  10 Pull Ups | Jumping Pull Ups
  10 "I Suck At This"
```

Perform a movement that you suck at. If it is already in the warm up, then perform 15 repetitions of that movement instead of 10. For instance, if you suck at sit ups, and there are sit ups in the warm up, do 3 rounds of 15 instead of 3 rounds of 10. If you suck at handstand push ups, perform 3 rounds of 10 handstand push ups (or their progression) and add that to the warm up.

## WOD

### Étudiant — Beginner

15-7-9 repetitions for time of:

```wod
(15-7-9)
  18in Box Jumps // sub tuck jumps if no box
  Jumping Pull Ups
  Alternating Leg Lunges
```

### Avancée — Advanced

21-15-9 repetitions for time of:

```wod
(21-15-9)
  24in Box Jumps // sub tuck jumps if no box
  Pull Ups
  Alternating Leg Lunges
```

### Traceur — Expert

21-15-9 repetitions for time of:

```wod
(21-15-9)
  30in Box Jumps // sub tuck jumps if no box
  Pull Ups
  Alternating Leg Pistols
```
