## WOD

WOD 25 is a Tabata protocol circuit using four exercises (Air squats, push ups, sit ups, and pull ups). The format requires 20 seconds of work followed by 10 seconds of rest for each set. There is no rest between the individual exercises. Track all total repetitions completed across all sets to calculate your score.

### Étudiant

Complete a half Tabata circuit (4 sets per exercise):

```wod
(4) :20 Air Squats ? / :10 Rest
(4) :20 Push Ups ? / :10 Rest
(4) :20 Sit Ups ? / :10 Rest
(4) :20 Pullups ? / :10 Rest
```

### Avancée

Complete the full Tabata circuit (8 sets per exercise), using necessary substitutions while maintaining form:

```wod
(8) :20 Air Squats ? / :10 Rest
(8) :20 Push Ups ? / :10 Rest
(8) :20 Sit Ups ? / :10 Rest
(8) :20 Pullups ? / :10 Rest
```

### Traceur

Complete the full Tabata circuit (8 sets per exercise), maintaining correct form and full range of motion for every movement:

```wod
(8) :20 Air Squats ? / :10 Rest
(8) :20 Push Ups ? / :10 Rest
(8) :20 Sit Ups ? / :10 Rest
(8) :20 Pullups ? / :10 Rest
```
