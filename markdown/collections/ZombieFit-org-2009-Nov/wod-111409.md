## Warm Up

Run 1/4 mile | bike 2 miles | row 500m followed by:

```wod
(3)
  50m Quadrupedal Movement
  10 Air Squats
  10 Sit Ups
  10 Pull Ups // may sub Push Ups
```

## WOD

### Étudiant — Beginner

Practice the press movement. Use a broomstick or PVC pipe. Pause after each rep and correct form. This is NOT a conditioning WOD.

```wod
50 Shoulder Press // PVC or broomstick
50 Push Press // PVC or broomstick
50 Push Jerk // PVC or broomstick
```

### Avancée — Advanced

If unfamiliar with any movement, perform at the Étudiant level. Keep weight very light if you require a spot; focus on form.

```wod
Shoulder Press sets: 5, 5, 3, 3, 1, 1, 1
Push Press sets: 5, 5, 3, 3, 1, 1, 1
Push Jerk sets: 5, 5, 3, 3, 1, 1, 1
```

### Traceur — Expert

Familiar with all three movements; work towards a one-rep max.

```wod
Shoulder Press sets: 5, 5, 3, 3, 1, 1, 1
Push Press sets: 5, 5, 3, 3, 1, 1, 1
Push Jerk sets: 5, 5, 3, 3, 1, 1, 1
```

## Cool Down

(3)
  25 Push Ups
