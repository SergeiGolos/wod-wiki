## Warm Up

Warm up options include running 1/4 mile, biking 2 miles, or rowing 500m. This should be followed by:

```wod
(3) :? 10 Sit Ups / 10 Push Ups / 10 Air Squats / 10 Pull Ups (or Handstand Push Ups)
```

## WOD

WOD 33 is a 21-15-9 for time challenge.

### Étudiant

```wod
(21-15-9)
  12 inch Box Jumps
  Push Ups
  Sit Ups
```

### Avancée

```wod
(21-15-9)
  24 inch Box Jumps
  Push Ups
  Elbows to Knees
```

### Traceur

```wod
(21-15-9)
  36 inch Box Jumps
  Push Ups
  Elbows to Knees
```
*Note: If needed, subparkour squats can be used for box jumps.*

## Cool Down

Perform a 100m quadrupedal movement.
