## Warm Up

Warm up options (Choose one):
*   Run 1/4 mile
*   Bike 1/2 miles
*   Row 250m

Followed by:
```wod
(3)
  50m Quadrupedal Movement
  10 Air Squats
  10 Sit Ups
  10 Jumping Pull Ups (Substitute with Push Ups if no pull up bar)
```

## WOD

### Étudiant
```wod
(30)
  1 Burpee
  1 Jumping Pull Up
```

### Avancée
```wod
30 Burpee Pull Ups
```

### Traceur
```wod
50 Burpee Pull Ups
```

## Cool Down

Perform 50 Sit Ups.
