## Warm Up

Run 1/4 mile OR row 250m OR bike 1 mile.

```wod
(3)
  25m Quadrupedal Movement
  10 Push Ups
  10 Sit Ups
  10 Pull Ups or Jumping Pull Ups
```

## WOD

### Étudiant — Beginner

Perform the following movements for time:

```wod
20 Push Ups
40 Jumping Jacks
20 Sit Ups
40 Jumping Jacks
20 Air Squats
40 Jumping Jacks
20 Jumping Pull Ups
Rest 1:00
200m Sprint
```

### Avancée — Advanced

Perform the following movements for time:

```wod
25 Push Ups
25 Jumping Pull Ups
25 Sit Ups
25 Jumping Pull Ups
25 Air Squats
25 Jumping Pull Ups
25 5-Foot Precisions
25 Jumping Pull Ups
Rest 1:00
200m Sprint
```

### Traceur — Expert

Perform the following movements for time:

```wod
25 Push Ups
25 Pistols Left Leg
25 Handstand Push Ups
25 Pistols Right Leg
25 Pull Ups
25 Knees to Elbows
25 5-Foot Precisions
25 Jumping Pull Ups
Rest 1:00
200m Sprint
Rest 1:00
200m Sprint
```
