## WOD
### Étudiant
Tabata circuit focusing on squats and pushups. (8 rounds total for the pair).
```wod
(8) :20 ? Air Squats / :10 Rest
(8) :20 ? Pushups / :10 Rest
```

### Avancée
Tabata circuit including squats, pushups, situps, and pullups. (8 rounds total for the quartet).
```wod
(8) :20 ? Air Squats / :10 Rest
:20 ? Pushups / :10 Rest
:20 ? Situps / :10 Rest
:20 ? Pullups / :10 Rest
```

### Traceur
Tabata circuit including squats, pushups, situps, pullups, and 5 foot precisions. (8 rounds total for the quintet).
```wod
(8) :20 ? Air Squats / :10 Rest
:20 ? Pushups / :10 Rest
:20 ? Situps / :10 Rest
:20 ? Pullups / :10 Rest
:20 ? 5ft Precisions / :10 Rest
```

## Notes
*   Each repetition is 1 point. Total up your points and post them to comments.
*   There is no rest in-between sets.
