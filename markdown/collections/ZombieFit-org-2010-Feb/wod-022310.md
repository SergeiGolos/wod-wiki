## WOD

### Étudiant

```wod
(20) :60 EMOM / +5 Pushups / +5 Situps / +5 Air Squats
```

*Notes: Perform 5 push ups, 5 sit ups and 5 air squats on the minute for 20 minutes.*

### Avancée

This is a four-stage circuit performed sequentially (no rest between stages).

**Stage 1:**
```wod
(5) :60 EMOM / +5 Pushups / +5 Situps / +5 Air Squats
```

**Stage 2:**
```wod
(5) :60 EMOM / +10 Pushups / +5 Situps / +5 Air Squats
```

**Stage 3:**
```wod
(5) :60 EMOM / +10 Pushups / +10 Situps / +5 Air Squats
```

**Stage 4:**
```wod
(5) :60 EMOM / +10 Pushups / +10 Situps / +10 Air Squats
```

### Traceur

This stage follows the same four-stage structure as Avancée, but requires an additional movement penalty after every round.

**Stage 1:**
```wod
(5) :60 EMOM / +5 Pushups / +5 Situps / +5 Air Squats / +2 Burpees
```

**Stage 2:**
```wod
(5) :60 EMOM / +10 Pushups / +5 Situps / +5 Air Squats / +2 Burpees
```

**Stage 3:**
```wod
(5) :60 EMOM / +10 Pushups / +10 Situps / +5 Air Squats / +2 Burpees
```

**Stage 4:**
```wod
(5) :60 EMOM / +10 Pushups / +10 Situps / +10 Air Squats / +2 Burpees
```

*Notes: If you cannot complete all movements within the minute timeframe, you must stop the exercise, complete 2 burpees, and then move on to the next minute’s repetitions.*
