## Warm Up

Run 1/4 mile OR bike 2 miles OR row 250m.

```wod
(3)
  50m Quadrupedal Movement
  5 Five-Foot Precisions
  Push Ups
  Double Unders
```

## WOD

This WOD is NOT for time. Complete each set of exercises as quickly as possible, but rest as needed between sets. For example, complete the first set of pull-ups as quickly as possible, rest as needed, then begin the set of burpees. Continue in this fashion for all levels.

### Étudiant — Beginner

```wod
(21-15-9)
  Pull Ups
  Burpees
```

### Avancée — Advanced

```wod
(21-15-9)
  Weighted Thrusters
  Pull Ups
  Burpees
```

### Traceur — Expert

```wod
(21-15-9)
  Weighted Thrusters
  Pull Ups
  Burpees
  Knees To Elbows
```

A thruster is a front squat that goes into a full press. Pick a challenging weight that you can perform at least 5 times continuously — warm up the movement; don’t just pick an arbitrary weight and start the WOD.
