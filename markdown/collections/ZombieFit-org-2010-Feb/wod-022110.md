## WOD

This WOD is not for time. Complete each set of exercises as quickly as possible, resting as much as needed between sets. When starting the next set, complete all reps as fast as you can.

### Étudiant

```wod
(21-15-9) Pullups / Burpees
```

### Avancée

*A thruster is a front squat that goes into a full press.* Pick a challenging weight (bar or dumbbell) that you can perform at least 5 times continuously. Warm up the movement thoroughly before starting.

```wod
(21-15-9) Weighted Thrusters ?lb / Pullups / Burpees
```
