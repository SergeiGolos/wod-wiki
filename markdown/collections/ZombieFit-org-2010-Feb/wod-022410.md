## Warm Up

Run 1/4 mile | bike 2 miles | row 250m then:

```wod
(3)
  50m Quadrupedal Movement
  5 Five-Foot Precisions
  Push Ups
  Double Unders
```

## WOD

### Étudiant — Beginner

Perform the following movements for time:

```wod
(21-15-9)
  Push Ups
  Jumping Jacks
  Air Squats
```

### Avancée — Advanced

Perform the following movements for time. In between each set perform 25m of quadrupedal movement. Finish the WOD with a 25m quadrupedal movement.

```wod
(21-15-9)
  Push Ups
  25m Quadrupedal Movement
  Jumping Jacks
  25m Quadrupedal Movement
  Air Squats
  25m Quadrupedal Movement
```

### Traceur — Expert

Perform the following movements for time. In between each set perform 25m of quadrupedal movement. Finish the WOD with a 25m quadrupedal movement.

```wod
(21-15-9)
  Handstand Push Ups
  25m Quadrupedal Movement
  Jumping Jacks
  25m Quadrupedal Movement
  Air Squats
  25m Quadrupedal Movement
```
