# Sprint Introduction

**Type**: Speed Development  
**Duration**: 45 minutes  
**Difficulty**: Beginner  

### Description
This workout introduces young swimmers to sprinting concepts with short, fast efforts followed by complete recovery. The focus is on explosive starts and maintaining stroke length at higher speeds. Full rest intervals ensure swimmers can perform at their best for each effort while learning the difference between sprint and distance swimming.

```wod
(4) Warmup
  50m Freestyle
  :30 Rest

(6) Sprint Set
  25m Freestyle Sprint
  1:00 Rest

(4) Dive Practice
  15m Dive Entry
  :45 Rest

(4) Finish Sprints
  25m Freestyle Sprint
  :45 Rest

100m Cooldown
```

### Breakdown
- **Warm-up**: 200m progressive effort to prepare muscles
- **Sprint Set**: 150m total at maximum effort with full recovery
- **Dive Practice**: Teaches proper start technique and streamlining
- **Finish Sprints**: Final high-intensity efforts to simulate race finishes
- **Cooldown**: 100m easy swimming for recovery

### Collectible Metrics
- Sprint times for each 25m effort
- Improvement from first to last sprint (fatigue resistance)
