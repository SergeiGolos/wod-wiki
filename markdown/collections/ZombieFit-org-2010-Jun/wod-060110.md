## Warm Up

Run 1/4 mile | row 250m | bike 2 miles.

3 rounds of:
- 25m quadrupedal movement
- 10 sit-ups
- 10 pull-ups | jumping pull-ups
- 10 push-ups
- 10 air squats

## WOD

### Étudiant — Beginner

Perform for time, in as few sets as possible (go to failure on each movement).

```wod
50 Jumping Pull Ups
50 Sit Ups
50 Air Squats
50 Push Ups
```

### Avancée — Advanced

Perform for time, in as few sets as possible (go to failure on each movement).

```wod
75 Jumping Pull Ups
75 Sit Ups
75 Air Squats
75 Push Ups
```

### Traceur — Expert

Perform for time, in as few sets as possible (go to failure on each movement).

```wod
100 Jumping Pull Ups
100 Sit Ups
100 Air Squats
100 Push Ups
```
