## Warm Up

Warm up options:
*   Run ?mile (or 1/4 mile)
*   Row ?m (or 250m)
*   Bike ?miles (or 2 miles)

Perform 3 rounds of:
*   25m quadrupedal movement
*   10 sit ups
*   10 pull ups (or jumping pull ups)
*   10 push ups
*   10 air squats

## WOD

### Étudiant

Perform for time, and in as few sets as possible (i.e. go to failure on each movement), complete:
```wod
50 Jumping Pullups
50 Sit Ups
50 Air Squats
50 Push Ups
```

### Avancée

Perform for time, and in as few sets as possible (i.e. go to failure on each movement), complete:
```wod
75 Jumping Pullups
75 Sit Ups
75 Air Squats
75 Push Ups
```

### Traceur

Perform for time, and in as few sets as possible (i.e. go to failure on each movement), complete:
```wod
100 Jumping Pullups
100 Sit Ups
100 Air Squats
100 Push Ups
```

*No cool down prescribed.*
