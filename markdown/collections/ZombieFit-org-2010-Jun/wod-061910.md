## Warm Up

Run 1/4 mile OR row 250m OR bike 2 miles.

3 rounds of:
- 25m quadrupedal movement
- 10 push-ups
- 10 air squats
- 10 pull-ups or jumping pull-ups

## WOD

### Étudiant — Beginner

```wod
(15-7-9)
  18 inch Box Jumps // sub with tuck jumps if you don't have something to jump to
  Jumping Pull Ups
  Alternating Leg Lunges
```

### Avancée — Advanced

```wod
(21-15-9)
  24 inch Box Jumps // sub with tuck jumps if you don't have something to jump to
  Pull Ups
  Alternating Leg Lunges
```

### Traceur — Expert

```wod
(21-15-9)
  30 inch Box Jumps // sub with tuck jumps if you don't have something to jump to
  Pull Ups
  Alternating Leg Pistols
```
