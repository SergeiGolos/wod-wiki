## Warm Up

Run 1/4 mile OR row 250m OR bike 2 miles.

```wod
3 rounds
  25m Quadrupedal Movement
  10 Dips
  10 Five-Foot Precisions
  10 Push Ups
```

## WOD

### Étudiant — Beginner

```wod
(3)
  50 Air Squats
  50 Jumping Pull Ups
  50 Sit Ups
```

### Avancée — Advanced

```wod
(5)
  50 Air Squats
  50 Jumping Pull Ups
  50 Sit Ups
```

### Traceur — Expert

At the end of each round choose: row 250m OR bike 1 mile OR run 300m.

```wod
(5)
  50 Air Squats
  50 Jumping Pull Ups
  50 Sit Ups
  250m Row OR 1mile Bike OR 300m Run
```

## Cool Down

```wod
Tabata Push Ups
```
