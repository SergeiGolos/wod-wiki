## Warm Up

```wod
(3)
  10 Push Ups
  10 Sit Ups
  10 Air Squats
  10 Jumping Jacks
```

## WOD

```wod
1mile Run | 2000m Row | 5mile Bike
```

## Cool Down

```wod
5:00 Hold Raised Push Up Position
```

If you need to rest, do it for only 3 breaths and then get back into the raised push up position. Try to stay up for as long as possible.
