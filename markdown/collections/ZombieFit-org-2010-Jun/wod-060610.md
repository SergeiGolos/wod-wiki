## Warm Up

Run 1/4 mile OR row 250m OR bike 2 miles.

```wod
(3)
  10 Air Squats
  10 Push Ups
  10 Sit Ups
  10 Pull Ups or Jumping Pull Ups
  10 "I Suck At This" // movement you suck at
```

perform a movement that you suck at. If it is already in the warm up, then perform 15 repetitions of that movement instead of 10. For instance, if I suck at sit ups, and there are sit ups in the warm up, I’ll do 3 rounds of 15 instead of 3 rounds of ten. If I suck at handstand push ups, I’ll perform 3 rounds of 10 handstand push ups (or their progression) and add that to the warm up.

## WOD

### Étudiant — Beginner

Perform the following movements for time:

```wod
(15-7-9)
  18 inch Box Jumps // sub with Tuck Jumps if you don’t have a box
  Jumping Pull Ups
  Alternating Leg Lunges
```

### Avancée — Advanced

Perform the following movements for time:

```wod
(21-15-9)
  24 inch Box Jumps // sub with Tuck Jumps if you don’t have a box
  Pull Ups
  Alternating Leg Lunges
```

### Traceur — Expert

Perform the following movements for time:

```wod
(21-15-9)
  30 inch Box Jumps // sub with Tuck Jumps if you don’t have a box
  Pull Ups
  Alternating Pistols
```
