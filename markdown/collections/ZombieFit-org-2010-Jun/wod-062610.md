## Warm Up

Run 1/4 mile, bike 1/2 mile, row 250m.

```wod
(3)
  10 Air Squats
  10 Push Ups
  10 Sit Ups
  50ft Quadrupedal Movement
```

## WOD

### Étudiant — Beginner

For time, perform 21-15-9 repetitions of:

```wod
(21-15-9)
  Push Ups
  Jumping Jacks
  Jumping Pull Ups
```

### Avancée — Advanced

For time, perform 21-15-9 repetitions of:

```wod
(21-15-9)
  Burpees
  Pull Ups
```

### Traceur — Expert

For time, complete 21-15-9 repetitions of:

```wod
(21-15-9)
  Pull Up Burpees
  Air Squats
```
