## Warm Up

Run 1/4 mile OR bike 1 mile OR row 250m followed by:

3 rounds of:
- 10 Air Squats
- 10 Sit Ups
- 10 Push Ups
- 10 Pull Ups

## WOD

### Étudiant — Beginner

Substitute 5 Sit Ups for Jumping Pull Ups if no pull-up bar.

```wod
(10) :60 EMOM
  + 3 Push Ups
  + 5 Jumping Pull Ups
  + 9 Air Squats
```

### Avancée — Advanced

Substitute 7 Sit Ups for Pull Ups if no pull-up bar.

```wod
(15) :60 EMOM
  + 5 Push Ups
  + 7 Pull Ups
  + 9 Air Squats
```

### Traceur — Expert

Substitute 7 Sit Ups for Pull Ups if no pull-up bar.

```wod
(20) :60 EMOM
  + 5 Handstand Push Ups
  + 7 Pull Ups
  + 9 Air Squats
```

If you fail to complete the round in the given minute, stop the clock, perform 5 burpees, then start the clock and continue the next minute’s movements.
