## Warm Up

Run 1/4 mile | bike 1/2 mile | row 250 m.

May sub pull ups with jumping pull ups if needed.

```wod
(3)
  10 Air Squats
  10 Push Ups
  10 Sit Ups
  10 Air Squats
  10 Pull Ups
```

## WOD

### Étudiant — Beginner

Stand in place and mark the spot where your feet are. Then find a spot approximately 4 feet away. That spot is where you’ll be precisioning to for the WOD.

```wod
:60
  1 Precision
  2 Precision
  3 Precision
  4 Precision
  5 Precision
  // Continue increasing by 1 precision each minute as long as able
```

### Avancée — Advanced

Pick a precision length of ~85% of your max broad jump. If possible, precision over a gap or up to a ledge.

```wod
:60
  1 Precision (85% max broad jump)
  2 Precision (85% max broad jump)
  3 Precision (85% max broad jump)
  4 Precision (85% max broad jump)
  // Continue increasing by 1 precision each minute as long as able
```

### Traceur — Expert

Max effort every precision. If possible, precision over a gap or to a ledge.

```wod
:60
  1 Precision // max effort
  2 Precision // max effort
  3 Precision // max effort
  4 Precision // max effort
  // Continue increasing by 1 precision each minute as long as able
```

## Cool Down

In the smallest number of sets possible, complete 50 push ups and 50 pull ups. (May sub with chin ups, jumping pull ups, | L-sit pull ups depending on ability.)

```wod
50 Push Ups
50 Pull Ups
```
