## Warm Up

Run 1/4 mile or bike 1 mile or row 250m followed by:

3 rounds of:
- 10 Air Squats
- 10 Sit Ups
- 10 Push Ups
- 10 Pull Ups

## WOD

### Étudiant — Beginner

```wod
(3)
  400m Run
  15 Push Ups
```

### Avancée — Advanced

```wod
(3)
  400m Run
  25 Push Ups
```

### Traceur — Expert

```wod
(3)
  400m Run
  50 Push Ups
```
