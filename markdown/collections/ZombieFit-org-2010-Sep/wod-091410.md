## Warm Up

Run 1/4 mile OR row 250m OR bike 2 miles.

```wod
(3)
  25m Quadrupedal Movement
  10 Jumping Jacks
  10 Sit Ups
  10 Pull Ups / Jumping Pull Ups
  10 Air Squats
```

## WOD

### Étudiant — Beginner

```wod
(15-9-7)
  12 inch Box Jumps
  Push Ups
  Alternating Leg Lunges
```

### Avancée — Advanced

```wod
(21-15-9)
  24 inch Box Jumps
  Push Ups
  Lunges each leg
```

### Traceur — Expert

Use pistols in place of lunges if you can do them.

```wod
(21-15-9)
  24 inch Box Jumps
  Handstand Push Ups
  Lunges each leg
```
