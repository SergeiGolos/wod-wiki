## Warm Up

```wod
Run 1/4 mile OR Row 250m OR Bike 2 miles

(3)
  25m Quadrupedal Movement
  10 Push Ups
  10 Pull Ups or Jumping Pull Ups
  10 'I Suck At This' Movement
```

perform a movement that you suck at. If it is already in the warm up, then perform 15 repetitions of that movement instead of 10. For instance, if you suck at sit ups, and there are sit ups in the warm up, do 3 rounds of 15 instead of 3 rounds of 10. If you suck at handstand push ups, perform 3 rounds of 10 handstand push ups (or their progression) and add that to the warm up.

## WOD

### Étudiant — Beginner

```wod
(8) :20 Tabata
  Air Squats
1:00 Rest
(8) :20 Tabata
  Box Jumps 12in (sub Tuck Jumps)
1:00 Rest
(8) :20 Tabata
  5-Foot Precisions
1:00 Rest
(8) :20 Tabata
  Jumping Pull Ups
```

### Avancée — Advanced

```wod
(8) :20 Tabata
  Quadrupedal Movement
1:00 Rest
(8) :20 Tabata
  Box Jumps 24in
1:00 Rest
(8) :20 Tabata
  6-Foot Precisions
1:00 Rest
(8) :20 Tabata
  Pull Ups
```

### Traceur — Expert

```wod
(8) :20 Tabata
  Quadrupedal Movement
1:00 Rest
(8) :20 Tabata
  Box Jumps 30in
1:00 Rest
(8) :20 Tabata
  7-Foot Precisions
1:00 Rest
(8) :20 Tabata
  L-Sit Pull Ups
```

For the tabata QM, use each twenty second period to go as far as possible while performing the quadrupedal movement. Score yourself by tabulating feet traveled while QMing.
