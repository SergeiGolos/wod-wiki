October 2012 features 9 ZombieFit workouts built around for time efforts. Programming rotates through for time, intervals and amrap formats to keep sessions varied and progressive. Athletes can expect for time, intervals, amrap and max effort work across the training block.

Standout themes this month include high-intensity conditioning, parkour-specific skill work, density-based AMRAP sessions. The 9 sessions form a solid training block with steady volume and progressive demand.
