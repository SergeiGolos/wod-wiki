## Warm Up

Run 1/4 mile OR bike 1/2 mile OR row 250m followed by:

```wod
(3)
  50m Quadrupedal Movement
  10 Sit Ups
  10 Jumping Pull Ups // may sub with Push Ups if you don't have a pull up bar
```

## WOD

### Étudiant — Beginner

15-7-9 repetitions for time of:

```wod
(15-7-9)
  12 inch Box Jumps
  Push Ups
  Air Squats
  Jumping Pull Ups
```

### Avancée — Advanced

21-15-9 repetitions for time of:

```wod
(21-15-9)
  24 inch Box Jumps
  Push Ups
  Air Squats
  Pull Ups
```

### Traceur — Expert

21-15-9 repetitions for time of:

```wod
(21-15-9)
  30 inch Box Jumps
  Push Ups
  Air Squats
  L-Sit Pull Ups
```

If needed, sub box jumps with tuck jumps.
