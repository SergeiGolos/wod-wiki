## Warm Up

Run 1/4 mile OR bike 1/2 mile OR row 250m followed by:

```wod
(3)
  50m Quadrupedal Movement
  10 Sit Ups
  10 Jumping Pull Ups
```

May substitute jumping pull ups with push ups if you don't have a pull-up bar.

## WOD

### Étudiant — Beginner

30 rounds for time:

```wod
(30)
  1 Burpee
  1 Jumping Pull Up
```

### Avancée — Advanced

```wod
30 Burpee Pull Ups
```

### Traceur — Expert

```wod
50 Burpee Pull Ups
```

## Cool Down

```wod
50 Sit Ups
```
