## Warm Up

Run 1/4 mile | bike 1/2 mile | row 500m followed by:

```wod
(3)
  50m Quadrupedal Movement
  10 Air Squats
  10 Sit Ups
  10 Pull Ups
```

May substitute push-ups for pull-ups if you don't have a pull-up bar.

## WOD

### Étudiant — Beginner

Perform the following for time:

```wod
(15-9-7)
  5-Foot Precision Jumps
  Push Ups
```

### Avancée — Advanced

Perform the following for time:

```wod
(21-15-9)
  7-Foot Precision Jumps
  Push Ups
```

### Traceur — Expert

Perform the following for time:

```wod
(21-15-9)
  7-Foot Precision Jumps
  Handstand Push Ups
```

## Cool Down

```wod
100m Reverse Quadrupedal Movement
```
