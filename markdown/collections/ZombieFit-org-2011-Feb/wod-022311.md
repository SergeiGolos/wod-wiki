## Warm Up

Run 1/4 mile | bike 1/2 mile | row 250m followed by:

3 rounds of:
- 50m quadrupedal movement
- 10 sit-ups
- 10 jumping pull-ups (may sub with push-ups if you don't have a pull-up bar)

## WOD

### Étudiant — Beginner

```wod
(6)
  100m Run
  1:00 Rest
```

### Avancée — Advanced

```wod
(10)
  100m Run
  1:00 Rest
```

### Traceur — Expert

```wod
(10)
  100m Run
  :30 Rest
```

## Cool Down

Handstand hold against a wall to failure x 3.
