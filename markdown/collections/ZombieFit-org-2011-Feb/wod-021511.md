## Warm Up

Run 1/4 mile | bike 1/2 miles | row 250m, followed by:

```wod
(3)
  50m Quadrupedal Movement
  10 Sit Ups
  10 Jumping Pull Ups // may sub with Push Ups if you don’t have a pull up bar
```

## WOD

In as few sets as possible and for time, complete the following:

### Étudiant — Beginner

Complete the following for time, aiming to do so in as few sets (and as unbroken) as possible.

```wod
100 Lunges (50 each leg)
50 Push Ups
50 Air Squats
50 Sit Ups
```

### Avancée — Advanced

Complete the following for time, aiming to do so in as few sets (and as unbroken) as possible.

```wod
76 Lunges (38 each leg)
75 Push Ups
75 Air Squats
75 Sit Ups
```

### Traceur — Expert

Complete the following for time, aiming to do so in as few sets (and as unbroken) as possible.

```wod
100 Lunges (100 each leg)
100 Push Ups
100 Air Squats
100 Sit Ups
```

## Post

Post time and sets completed to comments.

**This WOD is a little different as you are being asked to not only complete it as quickly as possible, but to do so in as few sets as possible. I.e. You must complete the sets as fast you can with the minimal amount of rest, but also attempt to keep the sets as unbroken as you can. You should be going to failure on every set.**
