# Sequences

A timer line starts with `*` and a Duration. Rounds go in `(...)` at
the top. Add both and the chapter quest will flip.

```wod
```

> Tip: type `*:30 Rest` for a 30-second rest, then `(2 Rounds)\n10 KB`.
