# Yasso 800s — Marathon Speed Work

**Source:** Hal Higdon / Runners World (Bart Yasso method)  
**Feed:** running  
**Type:** Track Intervals  
**Duration:** ~50–60 minutes total

## Workout

**Warm-Up**
- 10 min easy jog (Z2 effort)
- 4× strides (20 sec fast, 40 sec walk)

**Main Set**
- 6–10 × 800m repeats
- Target pace: your marathon goal time in minutes → run 800s in that many minutes/seconds
  - Example: 4:00 marathon goal → run each 800 in 4:00
- Rest interval: equal to work interval (jog/walk 800m between each)

**Cool-Down**
- 10 min easy jog
- Calf raises + hip flexor stretch

## WOD Syntax

```wod
(Warmup)
  10:00 Run
  (4)
    :20 Strides
    :40 Walk

(6)
  800m Run
  800m Recovery Jog

(Cooldown)
  10:00 Run
```

## Notes

The Yasso 800 is a famous predictor workout. Start with 6 reps and build to 10 over the training cycle. If you can hit 10 × 800 at your goal pace 3 weeks before race day, you're ready.

**Progression:** Add 1 rep per week until you reach 10.
