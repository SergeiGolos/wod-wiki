# Hill Repeats — Strength Builder

**Source:** Pete Pfitzinger / Advanced Marathoning  
**Feed:** running  
**Type:** Hill Repeats  
**Duration:** ~45 minutes

## Workout

**Warm-Up**
- 15 min easy run to the hill

**Main Set**
- 8 × hill repeats
  - Hill: 200–300m at 4–6% grade
  - Effort: 5K race effort on the way up (hard but controlled)
  - Recovery: easy jog back down
- Rest between reps: only the jog back (roughly 90 sec)

**Cool-Down**
- 10 min easy jog home
- Hip flexor stretch, IT band stretch, calf stretch

## WOD Syntax

```wod
(Warmup)
  15:00 Run

(8)
  300m Hill Run
  :90 Recovery Jog

(Cooldown)
  10:00 Run
```

## Notes

Hill repeats build quad strength, improve running economy, and develop fast-twitch recruitment without the impact of flat-ground speed work. The downhill recovery doubles as eccentric quad loading.

**Alternative:** Treadmill at 5% incline if no hills available.
