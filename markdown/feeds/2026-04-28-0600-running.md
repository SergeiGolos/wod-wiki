# Easy Recovery Run — Hal Higdon Novice 5K, Week 1 Day 1

**Source:** Hal Higdon Novice 5K Training Plan  
**Feed:** running  
**Type:** Easy Run  
**Duration:** 20–25 minutes

## Workout

**Warm-Up**
- 5 min brisk walk

**Main Set**
- 20–25 min easy run at conversational pace
- You should be able to speak in full sentences
- Target effort: 60–65% max HR

**Cool-Down**
- 5 min walk
- Light hamstring + quad stretch, 30 sec each

## WOD Syntax

```wod
(Warmup)
  5:00 Walk

20:00 Run

(Cooldown)
  5:00 Walk
```

## Notes

This is your foundation run. The goal is time on feet, not speed. If you need to walk, walk — the Hal Higdon Novice plan has no shame in run/walk intervals early on.

**Pace guidance:** "If you're out of breath, you're going too fast."
