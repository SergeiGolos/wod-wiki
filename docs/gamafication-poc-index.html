01

The First Run — a scripted magic moment

Duolingo: value-before-signup → +20% next-day retention

Today the playground is open and editable — a real asset — but nothing directs the first five minutes: the visitor is told to “try editing any line,” with no checked outcome, no celebration, and no chain into running the timer. Proposal: turn the existing Zero-to-Hero roadmap into a live micro-quest — one instruction, one validated action, one celebration at a time, ending with a logged workout in under five minutes.

  • Open-ended prompt (“try editing any line”) — no defined success, most visitors do nothing
  • No feedback loop: edit → nothing happens → Run → nothing celebrates
  • Roadmap exists on the page but isn’t wired to the demo it sits next to
  • One instruction per step; compiler verifies the edit actually happened

  • Still account-free — sign-up offered only after the first logged workout
02

Chapter quests that actually check the work

Compiler-as-judge: runtime-validated, un-speed-clickable

Today quest cards show developer-facing copy — literally “Quest has no validation schema” — so chapters gate on clicks, not learning. Proposal: rewrite each of the 14 quests as a fix-the-workout puzzle (the Flexbox Froggy pattern): show a broken workout, escalate hints, and let the parser and timer runtime act as the judge.

03

Progress that only moves when the user does

LinkedIn progress bar → +55% completion · endowed progress 34% vs 19%

Today challenge cards announce “Challenge complete.” for things the visitor never did — landing on the page “completes” a challenge — which trains users to ignore the whole reward system. Proposal: one honest roadmap: step 1 pre-ticked as an endowed head start, everything else checked only when the behavior is detected, summarized in a header ring that follows the user across pages.

04

A 3-question intake that branches the path

Fitbod adaptive users: +32% workouts/month, 70%+ active at 3 months

Today every visitor gets the same three doors — Open Journal, Browse Collections, New Workout Note — whether they’re a first-time CrossFitter or a coach programming for a class. Proposal: a 30-second intake (experience, equipment, goal) that reorders the quest map around the user’s actual intent — coaches hit Dialects early, home-gym beginners start with Basics and Protocols.

05

Day-one achievements — the retention shelf

Day-1 achievement unlocked: 33.4% vs 20.4% day-14 retention

Today there is no achievement system — the only wins are the self-firing challenge cards from section 03. Proposal: a shelf of five session-one achievements, each mapped to a real first-run event, designed so a brand-new visitor can unlock three in their first five minutes. Cross-app data puts day-one unlocks among the strongest retention levers that exist.

06

A weekly streak, not a daily one

Strava: weekly cadence fits training · freezes → +48% streak length

Today there is no cadence mechanic at all — nothing distinguishes a user who trained this week from one who vanished in May. Proposal: copy Strava’s deliberate choice for physical activity: the streak unit is a week with at least one logged workout (rest days are physiology, not failure), protected by earnable Streak Freezes. A separate, gentler daily quest streak serves the syntax-learning week.

07

The journal mints its own trophies

Self-competition ladder: motivation with zero other users required

Today logged results sit in the journal as plain text — a 38-second personal best looks identical to a mediocre day. Proposal: every logged result is checked against history; personal records fire celebrations and mint badges (first Murph, sub-10 Fran, 100 rounds logged). This is Strava’s KOM insight pointed inward: the most durable competitor is your past self.

08

Benchmark ladders & the shareable card

Strava segments: competition scoped to be winnable · leagues +25% completions

Today there is no competition and no way to show a result to anyone — the social loop is absent end to end. Proposal: canonical workouts (Fran, Cindy, Murph) become Strava-style segments: a ladder that defaults to you vs. your history and only widens to a matched cohort when enough comparable users exist. Every logged workout renders a shareable card whose link drops a friend straight into the playground with that workout pre-loaded.
